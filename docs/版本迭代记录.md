### 跨系统协同要点

| 涉及端 | 协同场景 | 本版本状态 | 说明 |
|:-----|:--------|:---------:|------|
| 平台端 ↔ 供应商端 | 商品供应链（分类/属性/SPU/SKU/供货价） | ✅ 完整 | 平台定义标准商品 → 供应商设置价格/库存 |
| 平台端 → 工程仓端 | 商品供应链（商品市场采购） | ✅ 完整 | 供应商→工程仓的采购链路 |
| 工程仓端 ↔ 供应商端 | 采购订单履约 | ✅ 完整 | 下单→确认→发货→收货→售后 |
| 工程仓端 → 施工方端 | 销售订单履约 | ✅ 完整 | 下单→确认→发货→收货→售后 |
| 平台端 ↔ 工程仓端 | 仓库/库存监控 | ✅ 完整 | 平台端全量只读查看 |
| 平台端 → 各端 | 商户入驻审核 | ✅ 完整 | 平台统一审核后各端可使用 |
| 平台端 → 各端 | 全平台订单监控 | ✅ 完整 | 平台端只读查看所有端订单 |
| 各端 → 平台端 | 财务/发票数据 | ✅ 完整 | 各端上传→平台端全量查看 |
