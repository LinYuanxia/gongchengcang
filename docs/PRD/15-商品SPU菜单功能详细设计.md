# 商品SPU菜单功能详细设计

## 一、功能概述

### 1.1 功能定位

商品SPU（Standard Product Unit，标准化产品单元）是商品中心的核心基础功能，负责管理商品的标准化信息。SPU是一组具有共同属性的商品的集合，通过SPU可以高效管理同类商品的通用信息，减少重复数据录入。

### 1.2 功能价值

- **标准化管理**：统一商品的标准化信息，确保数据一致性
- **减少重复**：同类商品的通用信息只需维护一次，提高效率
- **结构清晰**：SPU与SKU的层级关系使商品结构更加清晰
- **便于扩展**：基于SPU可以快速扩展不同规格的SKU

### 1.3 业务范围

- SPU的创建、编辑、删除、查询
- SPU基本信息管理（名称、编码、分类、单位等）
- SPU关联属性管理
- SPU与SKU的关联管理
- SPU的批量导入导出（可选）

## 二、业务规则

### 2.1 SPU编码规则

- **编码唯一性**：SPU编码在系统中唯一
- **编码格式**：建议使用分类缩写-商品缩写-序号格式，如GJ-LWG-001（钢筋-螺纹钢-001）
- **编码不可修改**：SPU编码创建后不可修改
- **编码长度限制**：建议不超过50字符

### 2.2 SPU命名规则

- **名称必填**：SPU名称必须填写
- **名称长度限制**：建议不超过100字符
- **名称唯一性**：同一分类下SPU名称不可重复
- **名称清晰**：SPU名称应准确描述商品特征

### 2.3 SPU分类规则

- **分类必填**：SPU必须关联到具体的商品分类
- **分类层级**：SPU只能关联到末级分类（三级分类）
- **分类变更**：SPU可以变更所属分类
- **分类删除约束**：有关联SPU的分类不能删除

### 2.4 SPU删除规则

- **删除前检查**：删除SPU前需要检查是否有关联的SKU
- **级联删除**：如果SPU下有关联的SKU，不允许直接删除
- **软删除**：建议采用软删除机制，保留删除记录
- **删除确认**：删除SPU需要二次确认

## 三、功能列表

### 3.1 SPU列表查询

#### 3.1.1 列表展示

##### 功能说明
分页查询SPU列表，支持多种搜索和筛选条件，方便用户快速定位所需的SPU信息。列表页面提供丰富的数据展示和操作功能，是SPU管理的主要入口。

##### 业务场景
- 日常查看和管理所有SPU信息
- 根据分类筛选查看特定类别的SPU
- 通过关键词搜索快速查找目标SPU
- 从分类管理页面跳转查看该分类下的SPU
- 对SPU进行批量操作（如批量导出）

##### 交互逻辑
1. **页面加载**：
   - 进入页面自动加载第一页SPU列表数据
   - 同时加载分类树数据用于筛选
   - 检查URL参数是否有categoryId，如有则自动筛选该分类下的SPU
   - 页面标题显示为"SPU列表"

2. **搜索功能**：
   - 搜索框支持输入SPU名称或编码进行模糊搜索
   - 支持实时搜索（输入后自动搜索）或点击搜索按钮搜索
   - 搜索后自动跳转到第一页
   - 搜索结果高亮显示匹配的关键词
   - 搜索框支持清除功能，点击清除按钮后恢复显示所有数据

3. **筛选功能**：
   - 分类级联选择器：支持选择一级、二级、三级分类
   - 选择分类后自动筛选该分类下的所有SPU
   - 支持清除筛选条件
   - 筛选后自动跳转到第一页
   - 分类选择器支持搜索功能，可快速定位分类

4. **数据展示**：
   - 表格形式展示SPU列表，包含主图、SPU编码、SPU名称、所属分类、计量单位、关联属性、SKU数量、创建时间等字段
   - 主图采用缩略图展示，鼠标悬停可放大查看
   - 关联属性以标签形式展示，超出数量时折叠显示
   - SKU数量显示为可点击的链接，点击可查看该SPU下的SKU列表
   - 所属分类显示分类路径（如：钢材 > 螺纹钢 > HRB400E）

5. **分页功能**：
   - 支持自定义每页显示条数（10/20/50/100条）
   - 显示当前页码、总页数、总记录数
   - 支持快速跳转到指定页
   - 支持上一页、下一页、首页、末页导航
   - 分页状态在页面刷新时保持

6. **排序功能**：
   - 支持按创建时间升序/降序排序
   - 支持按SPU编码升序/降序排序
   - 支持按SPU名称升序/降序排序
   - 默认按创建时间降序排序
   - 排序状态在页面刷新时保持

7. **列表操作**：
   - 支持单选/多选SPU进行批量操作
   - 提供批量导出功能
   - 提供批量删除功能（需二次确认）
   - 支持全选/反选功能

##### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 展示方式 |
|---------|---------|---------|---------|
| 主图 | SPU主图URL | VARCHAR | 图片展示，鼠标悬停可放大 |
| SPU编码 | SPU唯一编码 | VARCHAR | 文本展示 |
| SPU名称 | SPU名称 | VARCHAR | 主标题展示 |
| 所属分类 | 商品分类完整路径 | VARCHAR | 文本展示 |
| 计量单位 | 商品计量单位 | VARCHAR | 文本展示 |
| 关联属性 | SPU关联的属性名称 | JSON | 标签展示，超出5个时折叠 |
| SKU数量 | SPU下关联的SKU数量 | INT | 文本展示，可点击查看详情 |
| 创建时间 | SPU创建时间 | DATETIME | 文本展示，格式YYYY-MM-DD HH:mm:ss |

##### 操作按钮说明
- **详情**：查看SPU的完整信息，包括基本信息、关联属性、SKU列表等
- **编辑**：修改SPU的基本信息和关联属性
- **管理SKU**：跳转至SKU列表页面，自动筛选该SPU下的SKU
- **删除**：删除SPU（需二次确认，需检查是否有关联SKU）

##### 权限要求
- 平台管理员：可查看所有SPU，可执行所有操作
- 平台运营：可查看所有SPU，可执行所有操作
- 供应商：可查看所有SPU（只读），不可执行任何操作
- 工程仓：可查看所有SPU（只读），不可执行任何操作
- 施工方：可查看所有SPU（只读），不可执行任何操作

##### 异常处理
- 网络错误：显示错误提示，提供重试按钮
- 数据为空：显示空状态提示，引导用户创建SPU
- 搜索无结果：显示无搜索结果提示，建议调整搜索条件
- 筛选无结果：显示无筛选结果提示，建议调整筛选条件
- 分类不存在：提示分类不存在，引导重新选择分类

##### 界面元素
- 搜索框（支持输入SPU名称或编码）
- 分类级联选择器（支持一级、二级、三级分类）
- 商品列表表格
- 分页控件
- 批量操作按钮（批量导出、批量删除）
- 操作按钮（详情、编辑、管理SKU、删除）

### 3.2 新增SPU

#### 3.2.1 打开新增表单

##### 功能说明
打开新增SPU表单，填写SPU的基本信息和关联属性。

##### 业务场景
- 新商品上线时，先创建SPU标准化信息
- 同类商品新增时，基于已有SPU创建新SKU
- 商品标准化管理时，批量创建SPU

##### 交互逻辑
1. **入口**：
   - 点击SPU列表页右上角的"新增SPU"按钮
   - 从分类管理页面点击"添加SPU"按钮
   - 从SKU管理页面点击"新增SPU"按钮
   - 按钮位置固定在页面右上角，便于快速访问

2. **表单填写**：
   - 表单分为基本信息和关联属性两个部分
   - 基本信息包括：SPU名称、SPU编码、所属分类、计量单位、主图、商品描述
   - 关联属性包括：品牌、材质、产地、规格等（根据分类动态显示）
   - 带*号的为必填项
   - 表单实时校验，填写错误即时提示
   - 表单支持保存草稿功能（可选）

##### 界面元素
- 新增SPU按钮
- 新增SPU表单弹窗/抽屉

---

#### 3.2.2 表单填写

##### 功能说明
填写SPU的基本信息和关联属性。

##### 业务场景
- 创建新SPU时填写基本信息
- 填写SPU的关联属性
- 上传SPU主图

##### 交互逻辑
1. **SPU编码生成**：
   - 支持手动输入SPU编码
   - 支持系统自动生成SPU编码（根据分类编码+序号）
   - 编码唯一性校验，重复时自动提示并生成新编码
   - 编码格式校验，不符合规则时提示
   - 编码生成规则可配置

2. **分类选择**：
   - 使用级联选择器选择分类
   - 只能选择末级分类（三级分类）
   - 选择分类后，自动加载该分类下的关联属性
   - 分类变更后，清空已填写的关联属性
   - 分类选择器支持搜索功能，可快速定位分类

3. **关联属性管理**：
   - 根据所选分类自动显示该分类下的属性
   - 支持添加自定义属性
   - 支持删除属性（系统属性不可删除）
   - 属性值支持输入框、下拉框、多选框等多种形式
   - 属性值支持校验，如必填、格式等
   - 属性支持排序功能

4. **主图上传**：
   - 支持拖拽上传和点击上传
   - 支持上传jpg、png、gif格式图片
   - 图片大小限制（建议不超过5MB）
   - 上传前进行图片预览
   - 支持裁剪和压缩图片
   - 支持上传多张图片（可选，第一张为主图）

5. **商品描述**：
   - 支持富文本编辑器
   - 支持插入图片、表格、链接等
   - 支持格式化工具栏（加粗、斜体、下划线等）
   - 支持字数统计
   - 支持预览功能

##### 表单字段说明
| 字段名称 | 字段说明 | 数据类型 | 必填 | 展示方式 |
|---------|---------|---------|------|---------|
| SPU名称 | SPU名称 | VARCHAR | 是 | 输入框，长度1-100字符 |
| SPU编码 | SPU唯一编码 | VARCHAR | 是 | 输入框，长度1-50字符，系统内唯一 |
| 所属分类 | 商品分类 | VARCHAR | 是 | 级联选择器，只能选择末级分类 |
| 计量单位 | 商品计量单位 | VARCHAR | 是 | 下拉选择框，从系统预设单位中选择 |
| 主图 | SPU主图 | VARCHAR | 否 | 上传组件，支持jpg/png/gif格式 |
| 商品描述 | 商品详细描述 | TEXT | 否 | 富文本编辑器，长度0-2000字符 |
| 关联属性 | SPU关联的属性值 | JSON | 否 | 根据分类动态显示，支持自定义属性 |

##### 表单验证规则
- **SPU名称**：必填，长度1-100字符，同一分类下不可重复，支持中英文、数字、特殊字符
- **SPU编码**：必填，长度1-50字符，系统内唯一，支持字母、数字、下划线、横杠
- **所属分类**：必填，必须选择末级分类
- **计量单位**：必填，从系统预设单位中选择
- **主图**：可选，图片格式为jpg/png/gif，大小不超过5MB，建议尺寸800x800像素
- **商品描述**：可选，长度0-2000字符

##### 权限要求
- 平台管理员：可新增SPU
- 平台运营：可新增SPU
- 供应商：不可新增SPU
- 工程仓：不可新增SPU
- 施工方：不可新增SPU

##### 异常处理
- 网络错误：提交失败，保留表单数据，提示用户重试
- 编码重复：提示编码已存在，自动生成新编码建议
- 分类不存在：提示分类不存在，请重新选择
- 名称重复：提示同一分类下名称已存在，请修改名称
- 图片上传失败：提示上传失败，请检查图片格式和大小
- 属性值格式错误：提示属性值格式错误，请按照要求填写

##### 界面元素
- 表单输入框（SPU名称、SPU编码）
- 分类级联选择器
- 计量单位下拉框
- 主图上传组件
- 关联属性动态表单
- 富文本编辑器（商品描述）

修改SPU的基本信息和关联属性。

### 3.6 删除SPU

#### 3.6.1 单个删除SPU

##### 功能说明
删除单个SPU记录，需二次确认，删除前检查是否有关联SKU。

##### 业务场景
- SPU信息错误需要删除
- SPU不再使用需要删除
- 清理无效的SPU数据

##### 交互逻辑
1. 点击SPU列表页操作列的"删除"按钮
2. 弹出二次确认对话框
3. 确认对话框展示：
   - 标题：删除SPU
   - 提示信息：确定要删除该SPU吗？删除后将无法恢复
   - 检查是否有关联SKU，如有则提示：该SPU下有关联的SKU，无法直接删除，请先删除关联的SKU
4. 点击"确认"按钮，执行删除操作
5. 点击"取消"按钮，取消删除操作
6. 删除成功后，刷新列表数据

##### 删除规则
- **删除前检查**：删除前检查该SPU是否存在
- **级联检查**：删除前检查该SPU下是否有关联的SKU，如有则不允许删除
- **软删除**：建议采用软删除机制，保留删除记录
- **操作日志**：记录删除操作的操作人、操作时间、删除原因

##### 权限要求
- 平台管理员：可使用
- 平台运营：可使用
- 供应商：不可使用
- 工程仓：不可使用
- 施工方：不可使用

##### 异常处理
- SPU不存在：提示"SPU不存在"
- 有关联SKU：提示"该SPU下有关联的SKU，无法直接删除，请先删除关联的SKU"
- 删除失败：提示"删除失败，请重试"
- 网络错误：提示"网络错误，请检查网络连接"

##### 界面元素
- 删除按钮
- 删除确认对话框
- 确认按钮
- 取消按钮

#### 3.6.2 批量删除SPU

##### 功能说明
批量删除多个SPU记录，需二次确认，删除前检查是否有关联SKU。

##### 业务场景
- 批量清理无效的SPU数据
- 批量删除错误的SPU信息
- 批量删除不再使用的SPU

##### 交互逻辑
1. 在SPU列表页勾选多个SPU
2. 点击"批量删除"按钮，弹出二次确认对话框
3. 确认对话框展示：
   - 标题：批量删除SPU
   - 提示信息：确定要删除选中的 {n} 个SPU吗？删除后将无法恢复
   - 检查是否有关联SKU，如有则提示：选中的SPU中有 {m} 个SPU下有关联的SKU，无法删除
4. 点击"确认"按钮，执行批量删除操作
5. 点击"取消"按钮，取消删除操作
6. 删除完成后，显示删除结果：
   - 成功删除 {n} 个
   - 失败 {m} 个（列出失败原因）
7. 刷新列表数据

##### 删除规则
- **批量处理**：一次性处理多个SPU的删除
- **成功失败分离**：记录每个删除的成功/失败状态
- **部分失败处理**：部分删除失败时，显示失败详情，成功的部分正常删除
- **级联检查**：每个SPU删除前独立检查是否有关联SKU
- **操作日志**：记录批量删除操作的操作人、操作时间、删除数量

##### 权限要求
- 平台管理员：可使用
- 平台运营：可使用
- 供应商：不可使用
- 工程仓：不可使用
- 施工方：不可使用

##### 异常处理
- 未选择SPU：提示"请先选择要删除的SPU"
- 删除失败：显示失败详情
- 网络错误：提示"网络错误，请检查网络连接"

##### 界面元素
- 批量删除按钮
- 删除确认对话框
- 确认按钮
- 取消按钮
- 删除结果提示

**响应数据**：
```json
{
  "code": 200,
  "message": "success",
  "data": null
}
```

### 8.5 批量删除SPU接口

**接口地址**：`POST /api/spu/batch-delete`

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spuIds | Array | 是 | SPU ID数组 |

**响应数据**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "successCount": 2,
    "failCount": 1,
    "failList": [
      {
        "spuId": "SPU003",
        "reason": "该SPU下有关联的SKU，无法删除"
      }
    ]
  }
}
```
