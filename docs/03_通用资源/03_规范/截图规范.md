# 截图规范

## PRD 配图

| 图片类型 | 来源 | 用途 | 存放位置 |
|:--------|:-----|:----|:---------|
| 业务流程图 | Mermaid 导出 / Draw.io | 展示业务流程 | `各端/images/flowcharts/` |
| 线框图 | Axure / 手绘 | 低保真原型说明 | `各端/images/wireframes/` |
| 高保真原型 | Figma / 墨刀 | 视觉稿标注 | `各端/images/prototypes/` |

## 操作手册截图

| 要求 | 说明 |
|:----|:-----|
| 来源 | 系统**真实截图**，非设计稿 |
| 标注 | 关键操作区域用红色框线标注 |
| 错误截图 | 统一放入 `images/error/` 子目录 |
| 文件格式 | PNG，不超过 500KB |
