# Markdown 编写规范

## 标题

- `#` 一级标题：仅用在文件开头的文档标题
- `##` 二级标题：章节标题
- `###` 三级标题：子章节标题
- 最多使用四级标题 `####`

## 表格

- 表格前必须有表头行说明
- 使用对齐标记（`:---`、`:---:`、`---:`）

## 代码块

- 使用 ``` 代码块，指定语言（`md`、`ts`、`bash` 等）
- 路径/文件名应使用代码块包裹

## 链接

- 同目录文件：`[文件名](./文件名.md)`
- 跨目录文件：`[文件名](../目标目录/文件名.md)`

## 图片

- 语法：`![描述](路径)`
- 路径使用相对路径
- 截图必须添加描述性 Alt 文本
