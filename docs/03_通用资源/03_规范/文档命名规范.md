# 文档命名规范

## 目录命名

- `NN_中文名称/` — 使用**下划线**连接编号和中文名
- 编号范围：00~99
- 示例：`00_项目总览/`、`01_PRD/`、`02_系统操作手册/`

## 文件命名

- `NN-中文名称.md` — 使用**短横线**连接编号和中文名
- 编号范围：01~99，与目录编号独立
- 示例：`01-系统概览与架构.md`、`05-工作台功能设计.md`

## 图片命名

- `模块_页面_序号.png` — 下划线分隔，全小写英文
- 示例：`order_list_01.png`、`goods_add_modal.png`

## 版本后缀

- `v1.0`、`v1.1`、`v2.0` — 语义化版本号
