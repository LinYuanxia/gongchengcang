## 一、功能概述

### 1.1 功能定位

工作台是平台管理员登录后的**首页仪表盘**，提供平台核心运营指标的概览看板。让管理员和运营第一时间了解商户总数、订单情况、交易金额、商品SKU数等核心数据，辅助平台运营决策。

### 1.2 核心概念

| 概念 | 说明 | 示例 |
|:----|------|------|
| 全平台指标 | 所有端（供应商/工程仓/施工方）的数据汇总 | 总商户数、总交易额 |
| 今日动态 | 当日发生的关键业务变化 | 今日新增商户、今日订单数 |
| 趋势箭头 | 相比前一周期的变化趋势 | ↑15% 或 ↓3% |

### 1.3 目标用户

- **平台管理员**（核心用户）：全局了解平台整体运营状况
- **平台运营**：监控商品和订单的核心指标
- **平台财务**：关注交易金额和支付数据

### 1.4 模块范围

| 功能分类 | 主要功能 | 涉及角色 |
|:--------|---------|---------|
| 数据概览 | 数据概览看板（核心指标卡片+图表） | admin/operator/finance/service |

## 二、核心设计原则

> **工作台遵循"轻量概览、容错隔离"原则，不做深度数据分析。**

### 2.1 轻量概览原则

- 只展示关键运营指标（数字卡片+趋势+图表），不做明细数据下钻
- 默认展示"今日"数据，支持本周/本月时间切换
- 每个指标独立加载，互不影响（容错隔离）

### 2.2 时间维度原则

- 所有指标默认统计今日数据（00:00-23:59）
- 时间切换时并行重新请求所有统计接口
- 各指标使用统一的时间维度，不做不同时间范围的混合展示

### 2.3 只读不可操作原则

- 工作台所有数据为只读展示
- 点击指标卡片可配置跳转到对应模块列表页（不直接操作数据）

## 三、业务规则

### 3.1 数据展示规则

- **默认时间范围**：登录后默认展示"今日"数据
- **时间切换**：支持今日/本周/本月三个时间范围，切换时重新调取统计接口
- **自动刷新**：页面打开后每5分钟自动刷新一次（页面失焦时暂停轮询）
- **卡片跳转**：点击指标卡片中的数字可配置跳转到对应模块列表页
- **空数据**：指标为0时正常显示"0"，不做隐藏

### 3.2 指标预警规则

- 待审核商户数>0→数字变红色预警
- 待上架商品数>0→数字变橙色预警
- 预警类卡片可配置跳转到对应模块的筛选页

## 六、功能点详细设计

### 6.1 数据概览看板（P1）

#### 交互逻辑

1. 页面加载：默认选中"今日"时间范围 → 并行调用8个统计接口 → 渲染指标卡片网格 + 图表区
2. 切换时间：点击今日/本周/本月 → 时间参数变更 → 重新调用所有统计接口 → 卡片数据更新（骨架屏过渡）
3. 点击卡片数字：根据卡片类型跳转对应模块（如点击待审核商户→跳转商户管理按待审核筛选）
4. 自动刷新：页面激活状态下每5分钟自动刷新全部卡片（页面失焦时暂停轮询）

#### 原子字段定义

| 字段 | 必填 | 来源 | 校验规则 | 展示规则 | 默认值 |
|:----|:----|:----:|:----|:--------|:--------|:-----:|
| 时间范围 | 否 | 前端选择 | 三选一 | Radio按钮组 | 今日 |
| 商户总数 | 否 | 统计接口 | ≥0 | 数字卡片，大号字体 | 加载中... |
| 今日新增商户 | 否 | 统计接口 | ≥0 | 数字卡片+趋势箭头 | 加载中... |
| 订单总数 | 否 | 统计接口 | ≥0 | 数字卡片 | 加载中... |
| 今日订单数 | 否 | 统计接口 | ≥0 | 数字卡片+趋势箭头 | 加载中... |
| 交易总金额 | 否 | 统计接口 | ≥0 | 金额卡片，单位"元" | 加载中... |
| 商品SKU数 | 否 | 统计接口 | ≥0 | 数字卡片 | 加载中... |
| 待审核商户 | 否 | 统计接口 | ≥0 | >0时红色预警 | 加载中... |
| 待上架商品 | 否 | 统计接口 | ≥0 | >0时橙色预警 | 加载中... |

#### 边界情况覆盖

| 场景 | 处理逻辑 | 提示文案 |
|:----|:--------|---------|
| 单个统计接口超时 | 对应卡片显示"--"，其他卡片正常 | 不提示，卡片本身显示"--" |
| 全部接口失败 | 显示整页重试按钮 | "数据加载失败，请重试" |
| 时间切换后接口失败 | 保持上一次数据不变 | Toast提示"数据更新失败" |
| 所有指标为0 | 正常显示0 | - |
| 大面积预警 | 按规则标红/标橙 | - |

## 七、异常处理汇总表

| 异常场景 | 触发条件 | 处理方式 | 提示文案 |
|:--------|:--------|:--------|:--------|---------|
| 统计接口超时 | 响应>3s | 该卡片显示"--" | 无特殊处理 | 无提示（卡片直接显示"数据异常"） |
| 全部统计失败 | 所有卡片都失败 | 整页重试按钮 | 无特殊处理 | "数据加载失败，请重试" |
| 部分指标获取失败 | 接口返回错误 | 仅隐藏失败卡片 | 无特殊处理 | - |
