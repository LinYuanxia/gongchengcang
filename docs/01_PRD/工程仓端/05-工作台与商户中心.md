## 一、工作台功能设计（P1）

### 1.1 功能定位

工作台是工程仓登录后的**首页仪表盘**，提供关键运营指标的概览看板。让主管和采购员第一时间了解今日订单状态、采购金额、待收货情况、库存预警等核心数据，辅助决策。

### 1.2 目标用户

- **采购员**：关注今日订单数和采购金额
- **仓管员**：关注待收货数和库存预警
- **主管**：全局了解整体运营状况

### 1.3 数据概览看板

#### 功能说明
以卡片形式展示工程仓核心运营指标，支持时间范围切换。

#### 入口路径
登录后首页/左侧导航「工作台」

#### 交互逻辑
1. 页面加载：默认展示"今日"数据 → 调用统计接口 → 渲染指标卡片
2. 时间切换：点击今日/本周/本月 → 重新统计 → 更新卡片数据
3. 点击卡片：可配置快速跳转到对应模块列表（如点击待收货→跳转入库列表）
4. 自动刷新：每5分钟自动刷新一次（可配置）

#### 字段说明

| 字段名称 | 字段说明 | 组件类型 | 默认值 | 数据来源 |
|---------|---------|---------|:------:|---------|
| 时间范围 | 统计周期 | Radio | 今日 | 前端选择 |
| 今日订单数 | 采购+销售订单总数 | 数字卡片 | - | 统计接口 |
| 采购总金额 | 周期内采购订单金额合计 | 数字卡片 | - | 统计接口 |
| 待收货数 | 待收货的采购订单数 | 数字卡片(预警) | - | 统计接口 |
| 库存预警数 | 库存低于阈值的商品数 | 数字卡片(红色) | 0 | 统计接口 |
| 待处理销售单 | 待确认的销售订单数 | 数字卡片 | - | 统计接口 |

#### 业务规则
- 库存预警数>0→红色标红
- 待收货数>0→橙色标黄
- 点击库存预警→跳转商品中心（按预警筛选）
- 各指标数据为实时统计（非缓存）

#### 界面元素
- 顶部时间切换：今日/本周/本月 按钮组
- 指标卡片网格：2×3或3×2排列
- 每张卡片包含：图标+指标名称+数字+趋势（如有）
- 预警卡片：红色/橙色背景或边框
- 骨架屏：加载中显示卡片占位
- 刷新时间戳：底部显示"上次更新：HH:mm:ss"

#### 异常处理
| 异常场景 | 处理方式 | 提示信息 |
|---------|---------|---------|
| 统计接口超时 | 卡片显示"--" | 不影响其他卡片加载 |
| 全部接口失败 | 页面重试按钮 | "数据加载失败，请重试" |
| 无数据 | 卡片显示"0" | - |

#### 权限控制
采购员/仓管员/主管可见，财务/销售不可见。

## 二、商户中心功能设计

### 2.1 功能定位

商户中心管理工程仓的**主体信息和合同**。主体信息包含工程仓的企业资质、联系人等基础资料；合同管理展示与供应商/施工方签订的合同文档。

### 2.2 核心概念

| 概念 | 说明 |
|-----|------|
| 主体信息 | 工程仓的企业注册信息，含营业执照、法人等 |
| 合同 | 工程仓与供应商/施工方签订的合作协议 |

### 2.3 目标用户

- **主管**：编辑主体信息、查看合同
- **采购员/仓管员/财务**：查看主体信息
- **财务**：查看合同（关联收款/付款条款）

---

### 2.4 主体信息查看（P1）

#### 功能说明
展示工程仓的企业主体信息。

#### 入口路径
商户中心 → 主体信息

#### 交互逻辑
1. 页面加载：获取主体信息 → 渲染各字段（只读）
2. 主管身份→显示「编辑」按钮
3. 手机号中间4位脱敏展示

#### 字段说明

| 字段名称 | 字段说明 | 组件 | 校验规则 | 数据来源 |
|---------|---------|------|---------|---------|
| 主体名称 | 企业名称 | 文本 | 只读 | 商户接口 |
| 统一信用代码 | 18位信用代码 | 文本 | 18位，只读 | 商户接口 |
| 法人代表 | 企业法人 | 文本 | 只读 | 商户接口 |
| 联系人 | 联系人姓名 | 文本 | 只读 | 商户接口 |
| 联系电话 | 手机号 | 文本(脱敏) | 中间4位显示**** | 商户接口 |
| 详细地址 | 注册地址 | 文本 | 只读 | 商户接口 |
| 营业执照 | 图片 | Image | 只读（可放大） | 商户接口 |

#### 业务规则
- 所有角色可见
- 不存在的信息显示"-"
- 联系电话自动脱敏：138****1234

#### 界面元素
- 信息卡片：字段名+字段值（两列布局）
- 营业执照缩略图（可点击放大）
- 编辑按钮（仅主管可见）

#### 权限控制
所有角色可见。

---

### 2.5 主体信息编辑（P1）

#### 功能说明
编辑工程仓的联系信息和资质文件。

#### 入口路径
主体信息页 → 「编辑」按钮

#### 前置条件
当前用户=主管

#### 交互逻辑
1. 点击编辑→打开编辑弹窗/切换编辑模式
2. 修改字段→实时校验
3. 保存→提交更新接口→记录修改日志
4. 取消→恢复原始数据

#### 字段说明

| 字段名称 | 字段说明 | 组件 | 必填 | 校验规则 | 数据来源 |
|---------|---------|------|:----:|---------|---------|
| 联系人 | 联系人姓名 | Input | 否 | 2-20字 | 前端输入 |
| 联系电话 | 手机号 | Input | 否 | ^1[3-9]\d{9}$ | 前端输入 |
| 详细地址 | 注册地址 | Input | 否 | 最大100字 | 前端输入 |
| 营业执照 | 营业执照图片 | ImageUpload | 否 | jpg/png ≤5M | 前端文件 |

#### 业务规则
- 字段实时校验（输入即校验），错误时边框变红+提示
- 提交后记录修改日志（谁+什么时间+改了哪些字段）
- 营业执照支持替换上传

#### 界面元素
- 编辑弹窗/页面：表单+保存/取消按钮
- 保存按钮加载状态：防止重复提交

#### 异常处理
| 异常场景 | 处理方式 | 提示信息 |
|---------|---------|---------|
| 联系电话格式错误 | 实时标红 | "请输入正确的手机号" |
| 文件过大 | 前端拦截 | "营业执照图片不能超过5M" |
| 保存失败 | toast | "保存失败，请重试" |

#### 权限控制
仅主管可操作。

---

### 2.6 合同列表查看（P2）

#### 功能说明
查看工程仓与供应商/施工方签订的合作合同。

#### 入口路径
商户中心 → 合同列表

#### 交互逻辑
1. 状态Tab切换：生效/已过期
2. 时间范围筛选
3. 点击合同→查看合同详情（PDF预览）

#### 字段说明

| 字段 | 组件 | 数据来源 |
|------|------|---------|
| 状态Tab | Tab切换 | 前端固定 |
| 时间范围 | DatePicker | 前端输入 |
| 合同编号 | 文本 | 合同接口 |
| 合同名称 | 文本(链接) | 合同接口 |
| 签约方 | 文本 | 合同接口 |
| 生效日期 | 日期 | 合同接口 |
| 截止日期 | 日期 | 合同接口 |
| 状态(生效/已过期) | 标签 | 合同接口 |

#### 界面元素
- 状态Tab栏
- 合同列表表格：编号/名称/签约方/日期/状态

#### 权限控制
主管/财务可见。
