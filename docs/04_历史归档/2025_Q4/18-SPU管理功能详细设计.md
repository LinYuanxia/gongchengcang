> 版本：v1.0  
> 文档状态：评审中

---

## 一、功能概述

### 1.1 功能定位

SPU（Standard Product Unit，标准化产品单元）管理是平台商品体系的核心功能之一，负责定义商品的标准化信息。SPU是商品信息聚合的最小单位，是一组可复用、易检索的标准化信息的集合，该集合描述了一个商品的特性。SPU管理为商品SKU的创建提供基础支撑，是商品定价、库存管理、订单管理的前置条件。

### 1.2 核心概念

| 概念 | 说明 | 示例 |
|-----|------|------|
| SPU | 标准化产品单元，描述商品的通用特性 | 华为P40 |
| SKU | 库存保有单位，是SPU的具体实例 | 华为P40 8GB+128GB 黑色 |
| 规格属性 | 用于区分同一SPU下不同SKU的属性 | 颜色、尺寸、型号 |
| 规格组合 | 规格属性值的组合，用于生成SKU | 颜色:黑色 + 尺寸:8GB+128GB |

### 1.3 目标用户

- **平台管理员**：负责SPU的创建、编辑、删除等管理操作
- **平台运营人员**：负责SPU信息的维护和优化
- **供应商**：查看SPU信息，基于SPU创建供货关系
- **工程仓**：查看SPU信息，基于SPU进行采购
- **施工方**：查看SPU信息，基于SPU进行采购

## 二、业务规则

### 2.1 SPU基本信息规则

- **SPU名称**：SPU名称在系统中唯一
  - 最多支持100个字符
  - 建议包含品牌、型号等关键信息
  - 如"华为P40 5G手机"
- **SPU编码**：SPU编码在系统中唯一
  - 系统自动生成，支持手动输入
  - 建议使用英文、数字组合
  - 如"SPU202310010001"
- **所属分类**：SPU必须归属到具体的商品分类
  - 只能选择最末级分类
  - 分类变更时需要检查关联数据
- **计量单位**：SPU的计量单位
  - 支持预设单位（吨、米、个、件、套等）
  - 支持自定义单位
  - 如"个"、"件"、"吨"
- **商品主图**：SPU的主要展示图片
  - 支持JPG、PNG格式
  - 建议尺寸：800×800像素
  - 大小不超过2MB
- **商品相册**：SPU的详细展示图片
  - 最多支持9张图片
  - 支持JPG、PNG格式
  - 建议尺寸：800×800像素
  - 单张图片大小不超过2MB
- **商品描述**：SPU的详细描述信息
  - 最多支持500个字符
  - 支持换行和空格
- **商品备注**：SPU的备注信息
  - 最多支持500个字符
  - 支持换行和空格

### 2.2 规格属性规则

- **规格属性选择**：创建SPU时可选择多个规格属性
  - 从系统已启用的属性中选择
  - 至少选择一个规格属性
  - 规格属性名称不能重复
- **规格属性值**：每个规格属性需设置属性值
  - 至少设置一个属性值
  - 同一规格属性下的属性值不能重复
  - 如颜色属性可设置：黑色、白色、蓝色
- **规格组合**：系统根据规格属性值的笛卡尔积自动生成SKU
  - 如颜色有3个值，尺寸有3个值，生成3×3=9个SKU
  - 规格组合必须唯一
- **SKU数量限制**：单个SPU下的SKU数量建议控制在合理范围内
  - 一般建议不超过100个
  - 超过100个时性能会受影响

### 2.3 SKU生成规则

- **自动生成**：选择规格属性值后自动生成SKU
  - 系统根据规格属性值的笛卡尔积自动生成SKU
  - SKU编码和名称自动生成
- **SKU编码**：SKU编码自动生成，支持自定义
  - 默认规则：SPU编码 + "-" + 序号
  - 如SPU编码为"SPU001"，则SKU编码为"SPU001-0001"
  - 支持手动修改
- **SKU名称**：SKU名称自动生成，支持自定义
  - 默认规则：规格属性值组合
  - 如"黑色 8GB+128GB"
  - 支持手动修改
- **SKU主图**：SKU主图默认使用SPU主图
  - 支持单独上传SKU主图
  - SKU主图优先级高于SPU主图
- **价格设置**：每个SKU可设置独立的价格
  - 供货价：供应商供货价格
  - 销售价：平台销售价格
  - 支持批量设置价格

### 2.4 删除约束规则

- **SPU删除**：删除SPU前必须检查是否有关联数据
  - 检查是否有SKU：有SKU的SPU不能直接删除，需先删除SKU
  - 检查是否有供货关系：有供货关系的SPU不能直接删除，需先删除供货关系
  - 检查是否有订单：有订单的SPU不能直接删除
- **SKU删除**：删除SKU前必须检查是否有关联数据
  - 检查是否有供货关系：有供货关系的SKU不能直接删除，需先删除供货关系
  - 检查是否有订单：有订单的SKU不能直接删除
- **删除提示**：删除时根据校验结果给出明确的提示信息
  - 有SKU时提示："该SPU下有SKU，请先删除SKU"
  - 有供货关系时提示："该SPU下有供货关系，请先删除供货关系"

## 三、功能点详细设计

### 3.1 SPU列表展示

#### 功能说明
以列表形式展示SPU信息，支持搜索和筛选

#### 交互逻辑
1. 页面加载时显示所有SPU
2. 支持从分类管理页面跳转查看该分类下的SPU
3. SPU列表展示字段：
   - 主图：SPU主图，鼠标悬停可放大
   - SPU编码：SPU唯一编码
   - SPU名称：SPU名称
   - 所属分类：商品分类完整路径
   - 计量单位：商品计量单位
   - 关联属性：SPU关联的属性名称
   - SKU数量：SPU下关联的SKU数量
   - 创建时间：SPU创建时间
   - 操作按钮：详情、编辑、管理SKU、删除
4. 支持分页加载，当SPU数量较多时采用分页方式

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 展示方式 |
|---------|---------|---------|---------|
| 主图 | SPU主图URL | VARCHAR | 图片展示，鼠标悬停可放大 |
| SPU编码 | SPU唯一编码 | VARCHAR | 文本展示 |
| SPU名称 | SPU名称 | VARCHAR | 主标题展示 |
| 所属分类 | 商品分类完整路径 | VARCHAR | 文本展示 |
| 计量单位 | 商品计量单位 | VARCHAR | 文本展示 |
| 关联属性 | SPU关联的属性名称 | JSON | 标签展示，超出5个时折叠 |
| SKU数量 | SPU下关联的SKU数量 | INT | 文本展示，可点击查看详情 |
| 创建时间 | SPU创建时间 | DATETIME | 文本展示，格式YYYY-MM-DD HH:mm:ss |

#### 界面元素
- 搜索框（支持输入SPU名称或编码）
- 分类级联选择器（支持一级、二级、三级分类）
- SPU列表表格
- 分页控件
- 操作按钮（详情、编辑、管理SKU、删除）

### 3.2 搜索SPU

#### 功能说明
在搜索框输入关键词，支持模糊搜索SPU信息

#### 交互逻辑
1. 搜索框位于页面顶部，默认显示占位文字"搜索SPU名称/编码"
2. 输入关键词后点击搜索按钮触发搜索
3. 支持按回车键触发搜索
4. 搜索结果处理：
   - 匹配的SPU在列表中展示
   - 匹配的关键词高亮显示
   - 无匹配结果时显示"暂无搜索结果"提示
5. 清空搜索框时恢复SPU列表的原始展示状态

#### 字段说明
| 字段名称 | 字段说明 | 匹配规则 |
|---------|---------|---------|
| SPU名称 | 按SPU名称模糊匹配 | 包含关键词即匹配 |
| SPU编码 | 按SPU编码精确匹配或前缀匹配 | 完全匹配或前缀匹配 |

#### 界面元素
- 搜索输入框
- 搜索图标
- 搜索结果提示（无结果时显示）

---

### 3.3 筛选SPU

#### 功能说明
通过商品分类筛选SPU列表

#### 交互逻辑
1. 分类级联选择器位于搜索框右侧，默认显示占位文字"商品分类"
2. 点击级联选择器展开分类树
3. 分类树支持一级、二级、三级分类选择
4. 选择分类后自动筛选该分类下的SPU
5. 支持清除筛选，恢复显示所有SPU
6. 支持从URL参数自动筛选，如从分类管理页面跳转时自动筛选该分类下的SPU

#### 字段说明
| 字段名称 | 字段说明 | 匹配规则 |
|---------|---------|---------|
| 分类ID | 商品分类ID | 精确匹配，选择最末级分类 |

#### 界面元素
- 分类级联选择器
- 分类树
- 清除按钮

### 3.4 新增SPU

#### 功能说明
通过三步流程创建新的SPU，包括基本信息、规格属性、SKU定价

#### 交互逻辑
1. 点击"新增SPU"按钮跳转到新增SPU页面
2. **第一步：基本信息**：
   - 填写SPU名称*：必填项，最多100个字符
   - SPU编码：系统自动生成，支持手动输入
   - 选择所属分类*：必填项，选择最末级分类
   - 选择计量单位*：必填项，可选择预设单位或自定义
   - 上传商品主图：支持JPG、PNG格式，建议800×800像素
   - 上传商品相册：最多9张图片，支持JPG、PNG格式
   - 填写商品描述：最多500个字符
   - 填写商品备注：最多500个字符
   - 点击"下一步"按钮，校验表单完整性
3. **第二步：规格属性**：
   - 点击"添加规格"按钮添加规格属性
   - 选择规格名称：从系统已启用属性中选择，支持自定义
   - 设置规格值：每个规格属性至少设置一个属性值
   - 支持删除规格属性
   - 支持删除规格值
   - 实时显示SKU预览数量
   - 点击"下一步"按钮，校验规格属性完整性
4. **第三步：SKU定价**：
   - 查看SKU列表，显示所有生成的SKU
   - 编辑SKU编码：支持手动修改
   - 编辑SKU名称：支持手动修改
   - 上传SKU主图：支持单独上传SKU主图
   - 设置供货价：必填，大于0
   - 设置销售价：必填，大于0
   - 点击"批量设置价格"按钮：支持批量设置供货价和销售价
   - 点击"清空价格"按钮：清空所有SKU的价格
   - 点击"保存"按钮，提交保存
5. 保存成功后跳转到SPU列表页面，弹出成功提示："SPU创建成功"
6. 点击"取消"按钮或返回按钮，取消创建，不保存数据

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 约束 | 默认值 |
|---------|---------|---------|------|--------|
| SPU名称 | SPU名称，最多100字 | VARCHAR(100) | 必填 | - |
| SPU编码 | SPU唯一编码 | VARCHAR(50) | 必填，全局唯一 | 系统自动生成 |
| 所属分类 | 商品分类ID | VARCHAR(32) | 必填，最末级分类 | - |
| 计量单位 | 商品计量单位 | VARCHAR(20) | 必填 | - |
| 商品主图 | SPU主图URL | VARCHAR(255) | 可选 | - |
| 商品相册 | SPU相册图片URL列表 | JSON | 可选，最多9张 | - |
| 商品描述 | SPU描述信息 | VARCHAR(500) | 可选 | - |
| 商品备注 | SPU备注信息 | VARCHAR(500) | 可选 | - |

#### 界面元素
- 页面标题：新增SPU
- 步骤导航：基本信息 / 规格属性 / SKU定价
- 表单容器
- 步骤操作按钮：上一步、下一步、保存、取消
- 商品主图上传区域
- 商品相册上传区域
- 规格属性添加区域
- SKU列表表格
- 批量设置价格弹窗

### 3.5 编辑SPU

#### 功能说明
编辑已存在的SPU信息，操作与新增SPU一致

#### 交互逻辑
1. 在SPU列表的操作按钮中点击"编辑"按钮
2. 跳转到编辑SPU页面
3. 表单自动填充当前SPU的原有数据
4. **第一步：基本信息**：
   - 可修改：SPU名称、SPU编码、所属分类、计量单位、商品主图、商品相册、商品描述、商品备注
   - 点击"下一步"按钮，校验表单完整性
5. **第二步：规格属性**：
   - 可修改：规格属性、规格值
   - 修改规格属性后重新生成SKU
   - 点击"下一步"按钮，校验规格属性完整性
6. **第三步：SKU定价**：
   - 查看SKU列表，显示所有生成的SKU
   - 可修改：SKU编码、SKU名称、SKU主图、供货价、销售价
   - 点击"批量设置价格"按钮：支持批量设置供货价和销售价
   - 点击"清空价格"按钮：清空所有SKU的价格
   - 点击"保存"按钮，提交保存
7. 保存成功后跳转到SPU列表页面，弹出成功提示："SPU更新成功"
8. 点击"取消"按钮或返回按钮，取消编辑，不保存修改

#### 字段说明
同新增SPU字段说明

#### 界面元素
- 页面标题：编辑SPU
- 其他界面元素同新增SPU

| 商品主图 | SPU主图URL | VARCHAR | 图片展示，可放大 |
| 商品相册 | SPU相册图片URL列表 | JSON | 图片展示，可放大 |
| 商品描述 | SPU描述信息 | VARCHAR | 文本展示 |

| 更新时间 | SPU更新时间 | DATETIME | 文本展示，格式YYYY-MM-DD HH:mm:ss |

### 3.8 删除SPU

#### 功能说明
删除指定SPU，删除前需要校验是否有关联数据

#### 交互逻辑
1. 在SPU列表的操作按钮中点击"删除"按钮
2. 弹出确认删除对话框
3. 系统执行删除前校验：
   - **校验1：检查是否有SKU**
     - 如果有SKU，对话框显示警告信息："该SPU下有 {数量} 个SKU，请先删除SKU"
     - "确认"按钮置灰，不可点击
     - 显示"取消"按钮
   - **校验2：检查是否有供货关系**
     - 如果有供货关系，对话框显示警告信息："该SPU下有供货关系，请先删除供货关系"
     - "确认"按钮置灰，不可点击
     - 显示"取消"按钮
   - **校验3：检查是否有订单**
     - 如果有订单，对话框显示警告信息："该SPU下有订单，无法删除"
     - "确认"按钮置灰，不可点击
     - 显示"取消"按钮
   - **校验4：无关联数据**
     - 对话框显示确认信息："确定要删除SPU「{SPU名称}」吗？删除后不可恢复。"
     - 显示"确认"和"取消"按钮
4. 点击"确认"按钮：
   - 调用后端接口删除SPU
   - 删除成功后关闭对话框，刷新SPU列表
   - 弹出成功提示："SPU删除成功"
5. 点击"取消"按钮或弹窗外部关闭对话框，不执行删除

#### 字段说明
无额外字段，主要是校验逻辑

#### 界面元素
- 确认删除对话框
- 警告信息展示区域
- 确认按钮（校验通过时可用）

## 八、性能优化

### 8.1 性能优化点

1. **列表查询优化**：
   - 添加索引：spuName、spuCode、categoryId
   - 分页查询，避免一次性加载大量数据
   - 延迟加载图片，使用图片懒加载

2. **SKU生成优化**：
   - 前端预生成SKU，减少后端计算压力
   - 限制SKU数量，避免生成过多SKU

3. **图片优化**：
   - 图片压缩，减少文件大小
   - 使用CDN加速图片加载
   - 图片裁剪，统一尺寸

4. **缓存优化**：
   - 缓存分类树数据
   - 缓存SPU详情数据
   - 缓存属性列表数据
