> 版本：v1.0  
> 文档状态：评审中

---

## 一、功能概述

### 1.1 功能定位

SKU新增功能是商品管理的核心功能之一，用于为SPU（标准产品单位）创建具体的SKU（库存量单位）。每个SKU对应一个唯一的规格组合，包含价格、库存、图片等详细信息。

### 1.2 核心概念

| 概念 | 说明 | 示例 |
|-----|------|------|
| SPU | 标准产品单位，代表一类商品 | 普通硅酸盐水泥P.O42.5 |
| SKU | 库存量单位，代表具体规格 | 普通硅酸盐水泥P.O42.5 50kg/袋 红色 |
| 规格属性 | 商品的规格特征，如颜色、尺寸等 | 颜色、包装规格 |
| 规格组合 | 规格属性值的唯一组合 | 颜色:红色, 包装规格:50kg/袋 |
| SKU编码 | SKU的唯一标识 | SKU202401001 |

### 1.3 目标用户

- **平台管理员**：管理所有SKU信息
- **平台运营**：维护SKU信息
- **供应商**：查看自己供货的SKU信息
- **工程仓管理员**：查看SKU信息用于采购

## 二、业务规则

### 2.1 SKU基本信息规则

- **SKU编码**：SKU编码在系统中唯一
  - 格式：SKU + 年月日 + 序号
  - 如"SKU202401001"
  - 系统自动生成或手动输入
- **SKU名称**：SKU名称必填，最多支持200个字符
  - 如"普通硅酸盐水泥P.O42.5 50kg/袋 红色"
- **所属SPU**：每个SKU必须归属一个SPU
  - 从SPU列表中选择
  - 选择后自动填充SPU的单位
- **计量单位**：计量单位必填
  - 如"吨"、"米"、"个"、"件"等
  - 支持手动输入和预设选项
- **条形码**：条形码可选
  - 支持EAN-13、UPC等格式
  - 如"6901234567890"

### 2.2 规格属性规则

- **规格属性组合**：每个SKU的规格属性组合必须唯一
  - 同一SPU下不能有重复的规格组合
  - 如"颜色:红色, 尺寸:XL"和"颜色:红色, 尺寸:L"是不同的SKU
- **规格属性值**：规格属性值必须从SPU绑定的属性中选择
  - 不能自定义规格属性值
  - 如颜色属性只能选择红色、蓝色、绿色等值
- **规格展示**：规格属性以标签形式展示
  - 每个规格属性显示为一个标签
  - 如"颜色:红色"、"尺寸:XL"

### 2.3 图片上传规则

- **主图**：主图可选，最多上传1张
  - 支持JPG、PNG格式
  - 建议尺寸：800x800像素
  - 文件大小不超过5MB
- **商品相册**：商品相册可选，最多上传9张
  - 支持JPG、PNG格式
  - 建议尺寸：800x800像素
  - 文件大小不超过5MB
- **图片预览**：支持图片预览功能
  - 点击图片可放大查看
  - 支持左右切换查看多张图片

### 2.4 价格信息规则

- **供货价**：供货价可选，必须大于等于0
  - 保留2位小数
  - 如"420.00元/袋"
- **销售价**：销售价可选，必须大于等于0
  - 保留2位小数
  - 如"450.00元/袋"
- **成本价**：成本价可选，必须大于等于0
  - 保留2位小数
  - 如"400.00元/袋"
- **市场价**：市场价可选，必须大于等于0
  - 保留2位小数
  - 如"500.00元/袋"
- **价格校验**：销售价建议大于等于供货价
  - 如销售价低于供货价，给出警告提示
  - 但允许保存

### 2.5 状态管理规则

- **上架状态**：SKU状态分为上架和下架
  - 上架：正常使用，可参与采购、销售
  - 下架：暂停使用，不可参与采购、销售
- **状态变更**：支持手动切换状态
  - 上架→下架：确认后下架
  - 下架→上架：确认后上架

## 三、功能设计

### 3.1 SKU新增流程

```
选择SPU → 填写基本信息 → 选择规格属性 → 设置价格信息 → 上传图片 → 提交保存
```

### 3.2 页面设计

#### 3.2.1 页面布局

SKU新增页面采用抽屉式布局，包含以下区域：

1. **SPU选择区**：选择所属SPU
2. **基本信息区**：填写SKU编码、名称、条形码等
3. **价格信息区**：设置供货价、销售价、成本价、市场价
4. **图片上传区**：上传主图和商品相册
5. **规格属性区**：选择规格属性值

#### 3.2.2 交互设计

**SPU选择**：
- 点击SPU输入框，弹出SPU选择弹窗
- 支持搜索SPU名称/编码
- 支持分页查看SPU列表
- 选择后自动填充SPU的单位

**规格属性选择**：
- 选择SPU后，自动加载该SPU的规格属性
- 每个规格属性显示为一个下拉选择框
- 只能选择SPU已绑定的属性值
- 规格组合唯一性校验

**图片上传**：
- 支持拖拽上传
- 支持点击上传
- 支持图片预览
- 支持图片删除

**表单验证**：
- 必填项验证
- 格式验证（如SKU编码、条形码）
- 数值范围验证（如价格≥0）
- 规格组合唯一性验证

### 3.3 功能说明

#### 3.3.1 SPU选择功能

**功能描述**：从SPU列表中选择一个SPU作为SKU的所属SPU

**业务场景**：
- 创建SKU时必须指定所属SPU
- SKU继承SPU的分类、单位等信息

**交互逻辑**：
1. 点击SPU输入框，弹出SPU选择弹窗
2. 在弹窗中搜索或浏览SPU列表
3. 选择一个SPU，点击确定
4. 自动填充SPU的单位
5. 自动加载该SPU的规格属性

**字段说明**：
| 字段名 | 说明 | 必填 | 校验规则 |
|-------|------|------|---------|
| SPU名称 | SPU的名称 | 是 | - |
| SPU编码 | SPU的编码 | 否 | - |
| 分类 | SPU所属分类 | 否 | - |
| 单位 | SPU的计量单位 | 否 | - |
| 状态 | SPU的状态（启用/禁用） | 否 | - |

**权限要求**：
- 平台管理员：可选择所有SPU
- 平台运营：可选择所有启用的SPU
- 供应商：可选择自己供货的SPU
- 工程仓管理员：可选择所有启用的SPU

**异常处理**：
- 未选择SPU：提示"请选择所属SPU"
- SPU已禁用：提示"该SPU已禁用，请选择其他SPU"

#### 3.3.2 基本信息填写功能

**功能描述**：填写SKU的基本信息，包括SKU编码、名称、条形码、单位

**业务场景**：
- 创建SKU时必须填写基本信息
- SKU编码和名称用于识别SKU

**交互逻辑**：
1. 输入SKU编码
2. 输入SKU名称
3. 输入条形码（可选）
4. 确认单位（从SPU继承）

**字段说明**：
| 字段名 | 说明 | 必填 | 校验规则 |
|-------|------|------|---------|
| SKU编码 | SKU的唯一编码 | 是 | 唯一，最多50个字符 |
| SKU名称 | SKU的名称 | 是 | 最多200个字符 |
| 条形码 | SKU的条形码 | 否 | 最多50个字符 |
| 单位 | SKU的计量单位 | 是 | 最多20个字符 |

**权限要求**：
- 平台管理员：可编辑所有字段
- 平台运营：可编辑所有字段
- 供应商：可编辑所有字段
- 工程仓管理员：可编辑所有字段

**异常处理**：
- SKU编码重复：提示"SKU编码已存在，请重新输入"
- SKU名称为空：提示"请输入SKU名称"

#### 3.3.3 规格属性选择功能

**功能描述**：为SKU选择规格属性值，形成唯一的规格组合

**业务场景**：
- 每个SKU对应一个唯一的规格组合
- 规格组合用于区分不同的SKU

**交互逻辑**：
1. 选择SPU后，自动加载该SPU的规格属性
2. 为每个规格属性选择一个值
3. 系统自动校验规格组合的唯一性

**字段说明**：
| 字段名 | 说明 | 必填 | 校验规则 |
|-------|------|------|---------|
| 规格属性 | SPU绑定的规格属性 | 是 | - |
| 规格值 | 规格属性的可选值 | 是 | - |

**权限要求**：
- 平台管理员：可选择所有规格值
- 平台运营：可选择所有规格值
- 供应商：可选择所有规格值
- 工程仓管理员：可选择所有规格值

**异常处理**：
- 规格组合重复：提示"该规格组合已存在，请重新选择"
- 未选择所有规格属性：提示"请选择所有规格属性"

#### 3.3.4 价格信息设置功能

**功能描述**：设置SKU的价格信息，包括供货价、销售价、成本价、市场价

**业务场景**：
- 价格信息用于采购、销售等业务
- 不同价格适用于不同的业务场景

**交互逻辑**：
1. 输入供货价
2. 输入销售价
3. 输入成本价（可选）
4. 输入市场价（可选）
5. 系统自动校验价格范围

**字段说明**：
| 字段名 | 说明 | 必填 | 校验规则 |
|-------|------|------|---------|
| 供货价 | 供应商提供的价格 | 否 | ≥0，保留2位小数 |
| 销售价 | 面向客户的销售价格 | 否 | ≥0，保留2位小数 |
| 成本价 | SKU的成本价格 | 否 | ≥0，保留2位小数 |
| 市场价 | SKU的市场参考价格 | 否 | ≥0，保留2位小数 |

**权限要求**：
- 平台管理员：可编辑所有价格
- 平台运营：可编辑所有价格
- 供应商：可编辑供货价
- 工程仓管理员：可查看所有价格

**异常处理**：
- 价格小于0：提示"价格不能小于0"
- 销售价低于供货价：警告提示"销售价低于供货价，可能亏损"

#### 3.3.5 图片上传功能

**功能描述**：上传SKU的主图和商品相册图片

**业务场景**：
- 图片用于SKU的展示和识别
- 主图用于列表展示，商品相册用于详情展示

**交互逻辑**：
1. 点击上传按钮，选择图片文件
2. 上传主图（最多1张）
3. 上传商品相册图片（最多9张）
4. 支持图片预览和删除

**字段说明**：
| 字段名 | 说明 | 必填 | 校验规则 |
|-------|------|------|---------|
| 主图 | SKU的主要展示图片 | 否 | 最多1张，支持JPG、PNG |
| 商品相册 | SKU的多张展示图片 | 否 | 最多9张，支持JPG、PNG |

**权限要求**：
- 平台管理员：可上传所有图片
- 平台运营：可上传所有图片
- 供应商：可上传所有图片
- 工程仓管理员：可查看所有图片

**异常处理**：
- 图片格式错误：提示"图片格式不支持，请上传JPG或PNG格式"
- 图片大小超过限制：提示"图片大小不能超过5MB"

#### 3.3.6 提交保存功能

**功能描述**：提交SKU信息并保存到系统

**业务场景**：
- 填写完所有信息后提交保存
- 保存后SKU生效

**交互逻辑**：
1. 点击"确定"按钮
2. 系统自动校验所有必填项
3. 系统自动校验规格组合唯一性
4. 保存SKU信息
5. 提示保存成功
6. 关闭抽屉并刷新SKU列表

**字段说明**：
| 字段名 | 说明 | 必填 | 校验规则 |
|-------|------|------|---------|
| 所有必填项 | 表单中的必填字段 | 是 | - |
| 规格组合唯一性 | SKU的规格组合必须唯一 | 是 | - |

**权限要求**：
- 平台管理员：可提交保存
- 平台运营：可提交保存
- 供应商：可提交保存
- 工程仓管理员：不可提交保存

**异常处理**：
- 必填项为空：提示对应的必填项不能为空
- 规格组合重复：提示"该规格组合已存在，请重新选择"
- 保存失败：提示"保存SKU失败，请稍后重试"

## 五、接口设计

### 5.1 创建SKU接口

**接口地址**：`POST /api/sku`

**请求参数**：
```json
{
  "skuCode": "SKU202401001",
  "skuName": "普通硅酸盐水泥P.O42.5 50kg/袋 红色",
  "spuId": "spu001",
  "spuName": "普通硅酸盐水泥P.O42.5",
  "specs": {
    "颜色": "红色",
    "包装规格": "50kg/袋"
  },
  "barcode": "6901234567890",
  "unit": "袋",
  "mainImage": "https://example.com/image.jpg",
  "images": [
    "https://example.com/image1.jpg",
    "https://example.com/image2.jpg"
  ],
  "supplyPrice": 420.00,
  "salePrice": 450.00,
  "costPrice": 400.00,
  "marketPrice": 500.00
}
```

**响应参数**：
```json
{
  "success": true,
  "data": {
    "skuId": "sku001"
  },
  "message": "创建成功"
}
```

## 七、异常处理

### 7.1 业务异常

| 异常场景 | 异常提示 | 处理方式 |
|---------|---------|---------|
| SKU编码重复 | SKU编码已存在，请重新输入 | 阻止提交，提示用户重新输入 |
| 规格组合重复 | 该规格组合已存在，请重新选择 | 阻止提交，提示用户重新选择规格值 |
| SPU已禁用 | 该SPU已禁用，请选择其他SPU | 阻止选择该SPU |
| 价格小于0 | 价格不能小于0 | 阻止提交，提示用户重新输入 |
| 图片格式错误 | 图片格式不支持，请上传JPG或PNG格式 | 阻止上传，提示用户重新选择图片 |
| 图片大小超过限制 | 图片大小不能超过5MB | 阻止上传，提示用户重新选择图片 |
