# 工程材料管理采供一体化平台 - 规格属性功能详细设计

> 版本：v1.0  
> 文档状态：评审中  
> 所属模块：商品中心 - 规格属性

---

## 一、功能概述

### 1.1 功能定位

规格属性管理是平台商品体系的核心功能之一，负责定义商品的规格参数，用于区分同一商品SPU下的不同SKU。规格属性是商品SKU的关键组成部分，不同规格属性的组合会生成不同的SKU，为商品的库存管理、价格管理、订单管理提供基础支撑。

### 1.2 核心概念

| 概念 | 说明 | 示例 |
|-----|------|------|
| 规格组 | 规格的分类，如"颜色"、"尺寸"、"型号"等 | 颜色、尺寸、材质 |
| 规格值 | 规格组下的具体选项，如"红色"、"蓝色"等 | 红色、蓝色、大号、小号 |
| SKU规格 | 商品SKU的具体规格组合，由多个规格值组成 | 颜色:红色 + 尺寸:大号 |
| SKU | 库存保有单位，是规格属性组合的具体实例 | 红色大号商品 |

### 1.3 目标用户

- 平台管理员
- 平台运营人员
- 供应商（使用规格属性创建SKU）

## 二、业务规则

### 2.1 规格组规则

- **规格组名称**：规格组名称在系统中唯一
  - 如"颜色"、"尺寸"、"材质"等
  - 最多支持20个字符
- **规格组编码**：规格组编码在系统中唯一
  - 建议使用英文或拼音，如"color"、"size"
  - 自动生成或手动输入
- **输入类型**：规格组支持3种输入类型
  1. **单选**：从预设值中选择一个
  2. **多选**：从预设值中选择多个（不常用，建议用单选）
  3. **下拉选择**：从预设值中选择一个（推荐）
- **显示类型**：规格组在商品详情页的显示方式
  1. **文字**：纯文字显示
  2. **图片**：图片显示（如颜色可以显示颜色块）
  3. **文字+图片**：文字和图片混合显示
- **排序**：规格组按排序值排序显示

### 2.2 规格值规则

- **规格值名称**：同一规格组下的规格值名称唯一
  - 如"红色"、"蓝色"、"大号"、"小号"
  - 最多支持20个字符
- **规格值编码**：规格值编码在系统中唯一
  - 建议使用英文或拼音，如"red"、"blue"
  - 自动生成或手动输入
- **规格值图片**：部分规格值可以上传图片
  - 如颜色规格值可以上传颜色块图片
  - 支持JPG、PNG格式，大小不超过2MB
- **排序**：规格值按排序值排序显示
- **启用/禁用**：规格值可以单独启用或禁用
  - 禁用的规格值在创建SKU时不可选
  - 已使用该规格值的SKU不受影响

### 2.3 规格组合规则

- **组合唯一性**：同一商品SPU下的SKU规格组合必须唯一
  - 不能有两个SKU具有完全相同的规格组合
  - 如"颜色:红色 + 尺寸:大号"只能有一个SKU
- **组合完整性**：SKU必须包含所有必填规格组的规格值
  - 规格组设置为必填时，SKU必须选择该规格组的规格值
  - 选填规格组可以不选择
- **组合数量**：理论上可以生成无限个SKU，但建议控制在合理范围内
  - 一般建议单个SPU下的SKU数量不超过100个
  - 超过100个时性能会受影响

### 2.4 SKU生成规则

- **自动生成**：选择规格值组合后自动生成SKU
  - 选择所有规格值后，系统自动生成SKU
  - SKU编码自动生成
- **批量生成**：根据规格值的笛卡尔积批量生成SKU
  - 如颜色有3个值，尺寸有3个值，可以批量生成3×3=9个SKU
  - 支持选择性生成，可以只生成需要的组合
- **SKU编码**：SKU编码自动生成，支持自定义规则
  - 默认规则：SPU编码 + "-" + 规格值编码组合
  - 如SPU编码为"SPU001"，规格值为"red"和"large"，则SKU编码为"SPU001-red-large"
- **SKU名称**：SKU名称自动生成，支持自定义规则
  - 默认规则：SPU名称 + " " + 规格值名称组合
  - 如SPU名称为"T恤"，规格值为"红色"和"大号"，则SKU名称为"T恤 红色 大号"

### 2.5 价格和库存规则

- **价格独立**：每个SKU可以设置独立的价格
  - 不同规格组合的SKU可以有不同的价格
  - 如红色T恤比蓝色T恤贵10元
- **库存独立**：每个SKU有独立的库存
  - 不同规格组合的SKU库存独立管理
  - 如红色大号T恤库存100件，红色中号T恤库存50件
- **价格继承**：SKU价格可以继承SPU价格
  - 新建SKU时默认使用SPU价格
  - 可以手动修改SKU价格
- **库存联动**：SKU库存变动时同步更新SPU库存
  - SPU库存是所有SKU库存的总和
  - SKU库存增加时，SPU库存自动增加
  - SKU库存减少时，SPU库存自动减少

### 2.6 删除约束规则

- **规格组删除**：删除规格组前必须检查是否有关联数据
  - 检查是否有规格值：有规格值的规格组不能直接删除，需先删除规格值
  - 检查是否被商品使用：被商品使用的规格组不能直接删除，需先取消商品关联
- **规格值删除**：删除规格值前必须检查是否有关联数据
  - 检查是否被SKU使用：被SKU使用的规格值不能直接删除，需先取消SKU关联
- **删除提示**：删除时根据校验结果给出明确的提示信息
  - 有规格值时提示："该规格组下有规格值，请先删除规格值"
  - 被商品使用时提示："该规格被商品使用，请先取消商品关联"

## 三、功能点详细设计

### 3.1 规格组列表展示

#### 功能说明
以列表形式展示规格组，支持搜索和筛选

#### 交互逻辑
1. 页面加载时显示所有规格组
2. 规格组列表展示字段：
   - 规格组名称（最多显示10字，超出显示省略号）
   - 规格组编码（灰色小字）
   - 输入类型（单选/多选/下拉选择）
   - 显示类型（文字/图片/文字+图片）
   - 规格值数量（显示该规格组下的规格值数量，可点击展开查看）
   - 状态（启用/禁用标签）
   - 操作按钮（编辑/删除/启用/禁用/管理规格值）
3. 点击规格值数量标签可展开/折叠该规格组的规格值列表
4. 支持分页加载，当规格组数量较多时采用分页方式

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 展示方式 |
|---------|---------|---------|---------|
| 规格组名称 | 规格组的名称，最多20字 | VARCHAR | 主标题展示 |
| 规格组编码 | 规格组的唯一编码 | VARCHAR | 灰色小字展示 |
| 输入类型 | 规格组的输入类型 | TINYINT | 标签展示 |
| 显示类型 | 规格组的显示类型 | TINYINT | 标签展示 |
| 规格值数量 | 该规格组下的规格值数量 | INT | 数字展示，可点击 |
| 状态 | 规格组状态（启用/禁用） | TINYINT | 标签展示 |
| 排序 | 规格组的排序值 | INT | 不展示，用于排序 |
| 创建时间 | 规格组的创建时间 | DATETIME | 不展示 |
| 更新时间 | 规格组的更新时间 | DATETIME | 不展示 |

#### 界面元素
- 规格组列表容器
- 规格值展开/折叠按钮
- 操作按钮组
- 分页控件

---

### 3.2 搜索规格组

#### 功能说明
在搜索框输入关键词，支持模糊搜索规格组信息

#### 交互逻辑
1. 搜索框位于页面顶部，默认显示占位文字"搜索规格组名称或编码"
2. 输入关键词后实时触发搜索（支持防抖，延迟300ms）
3. 支持按回车键触发搜索
4. 搜索结果处理：
   - 匹配的规格组在列表中展示
   - 匹配的关键词高亮显示
   - 无匹配结果时显示"暂无搜索结果"提示
5. 清空搜索框时恢复规格组列表的原始展示状态

#### 字段说明
| 字段名称 | 字段说明 | 匹配规则 |
|---------|---------|---------|
| 规格组名称 | 按规格组名称模糊匹配 | 包含关键词即匹配 |
| 规格组编码 | 按规格组编码精确匹配或前缀匹配 | 完全匹配或前缀匹配 |

#### 界面元素
- 搜索输入框
- 搜索图标
- 清空按钮（输入内容后显示）
- 搜索结果提示（无结果时显示）

---

### 3.3 新增规格组

#### 功能说明
通过弹窗表单填写规格组信息，创建新的规格组

#### 交互逻辑
1. 点击"新增规格组"按钮弹出新增规格组弹窗
2. 表单包含以下字段：
   - **规格组名称**：
     - 必填项，带*号标识
     - 输入框，最多20个字符
     - 实时校验：输入时显示剩余字符数（如"10/20"）
     - 失焦时校验唯一性
   - **规格组编码**：
     - 必填项，带*号标识
     - 输入框，最多50个字符
     - 支持英文、数字、下划线
     - 自动生成按钮：点击自动生成编码（如"color"、"size"）
     - 失焦时校验唯一性
   - **输入类型**：
     - 单选按钮组，必填项，带*号标识
     - 选项：单选 / 多选 / 下拉选择
     - 默认选中"下拉选择"
   - **显示类型**：
     - 单选按钮组，必填项，带*号标识
     - 选项：文字 / 图片 / 文字+图片
     - 默认选中"文字"
   - **排序**：
     - 输入框，默认值为当前规格组的最大排序值+1
     - 只能输入正整数
     - 不填则使用默认值
   - **状态**：
     - 单选按钮组，默认选中"启用"
     - 选项：启用 / 禁用
     - 禁用状态时给出提示："禁用后创建商品时不可选"
3. 点击"确定"按钮：
   - 前端校验表单完整性
   - 调用后端接口创建规格组
   - 创建成功后关闭弹窗，刷新规格组列表
   - 弹出成功提示："规格组创建成功"
4. 点击"取消"按钮或弹窗外部关闭弹窗，表单数据不保存

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 约束 | 默认值 |
|---------|---------|---------|------|--------|
| 规格组名称 | 规格组名称，最多20字 | VARCHAR(20) | 必填，全局唯一 | - |
| 规格组编码 | 规格组编码 | VARCHAR(50) | 必填，全局唯一 | 自动生成 |
| 输入类型 | 输入类型 | TINYINT | 必填 | 3（下拉选择） |
| 显示类型 | 显示类型 | TINYINT | 必填 | 1（文字） |
| 排序 | 排序值，正整数 | INT | 可选 | 当前最大排序值+1 |
| 状态 | 规格组状态 | TINYINT | 必填 | 1（启用） |

#### 界面元素
- 弹窗标题：新增规格组
- 表单容器
- 规格组名称输入框（带字符计数）
- 规格组编码输入框（带自动生成按钮）
- 输入类型单选按钮组
- 显示类型单选按钮组
- 排序输入框
- 状态单选按钮组
- 确定按钮
- 取消按钮
- 表单验证提示

### 3.4 编辑规格组

#### 功能说明
编辑已存在的规格组信息，操作与新增规格组一致

#### 交互逻辑
1. 在规格组列表的操作按钮中点击"编辑"按钮
2. 弹出编辑规格组弹窗，表单字段与新增规格组一致
3. 表单自动填充当前规格组的原有数据
4. 字段规则：
   - 规格组名称：可修改，规则同新增
   - 规格组编码：可修改，规则同新增
   - 输入类型：不可修改（创建后输入类型不可修改）
   - 显示类型：可修改，规则同新增
   - 排序：可修改，规则同新增
   - 状态：可修改，规则同新增
5. 点击"确定"按钮：
   - 前端校验表单完整性
   - 调用后端接口更新规格组
   - 更新成功后关闭弹窗，刷新规格组列表
   - 弹出成功提示："规格组更新成功"
6. 点击"取消"按钮或弹窗外部关闭弹窗，修改不保存

#### 字段说明
同新增规格组字段说明，输入类型字段不可修改

#### 界面元素
- 弹窗标题：编辑规格组
- 其他界面元素同新增规格组
- 输入类型选择框禁用状态

---

### 3.5 管理规格值

#### 功能说明
管理规格组的规格值，包括新增、编辑、删除、启用/禁用规格值

#### 交互逻辑
1. 在规格组列表的操作按钮中点击"管理规格值"按钮
2. 跳转到规格值管理页面（或弹出弹窗）
3. 页面顶部显示规格组信息：
   - 规格组名称
   - 规格组编码
   - 输入类型
   - 显示类型
4. 规格值列表展示字段：
   - 规格值名称（最多显示10字，超出显示省略号）
   - 规格值编码（灰色小字）
   - 规格值图片（显示类型为图片或文字+图片时显示）
   - 排序
   - 状态（启用/禁用标签）
   - 操作按钮（编辑/删除/启用/禁用）
5. **新增规格值**：
   - 点击"新增规格值"按钮
   - 弹出新增规格值弹窗
   - 表单包含：
     - 规格值名称*：最多20个字符
     - 规格值编码*：自动生成或手动输入
     - 规格值图片：显示类型为图片或文字+图片时显示
     - 排序：默认值为当前规格值的最大排序值+1
     - 状态：默认选中"启用"
   - 点击"确定"按钮保存
6. **编辑规格值**：
   - 点击规格值操作列的"编辑"按钮
   - 弹出编辑规格值弹窗
   - 可修改：规格值名称、规格值编码、规格值图片、排序、状态
   - 点击"确定"按钮保存
7. **删除规格值**：
   - 点击规格值操作列的"删除"按钮
   - 弹出确认删除对话框
   - 校验是否被SKU使用
   - 被SKU使用的规格值不能删除，只能禁用
   - 未被使用的规格值可以删除
   - 点击"确认"按钮删除
8. **启用/禁用规格值**：
   - 点击规格值操作列的"启用/禁用"按钮
   - 弹出确认对话框
   - 点击"确认"按钮更新状态
   - 禁用的规格值在创建SKU时不可选
9. **规格值排序**：
   - 支持拖拽排序
   - 拖拽后自动重新计算排序值

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 约束 |
|---------|---------|---------|------|
| 规格值名称 | 规格值名称，最多20字 | VARCHAR(20) | 必填，同一规格组下唯一 |
| 规格值编码 | 规格值编码 | VARCHAR(50) | 必填，全局唯一 |
| 规格值图片 | 规格值图片URL | VARCHAR(255) | 可选 |
| 排序 | 排序值，正整数 | INT | 可选 |
| 状态 | 规格值状态 | TINYINT | 必填 |

#### 界面元素
- 规格组信息展示区域
- 规格值列表容器
- 新增规格值按钮
- 编辑规格值弹窗
- 删除规格值确认对话框
- 启用/禁用确认对话框

### 3.6 规格组启用/禁用

#### 功能说明
启用或禁用规格组，控制规格组在创建商品时是否可选

#### 交互逻辑
1. 在规格组列表的操作按钮中点击"启用"或"禁用"按钮
2. 弹出确认对话框
   - 启用时显示："确定要启用规格组「{规格组名称}」吗？启用后创建商品时可以选择。"
   - 禁用时显示："确定要禁用规格组「{规格组名称}」吗？禁用后创建商品时不可选，已使用该规格组的商品不受影响。"
3. 点击"确认"按钮：
   - 调用后端接口更新规格组状态
   - 同时更新该规格组下所有规格值的状态（可选，或保持规格值状态不变）
   - 更新成功后关闭对话框，刷新规格组列表
   - 弹出成功提示："规格组状态更新成功"
4. 点击"取消"按钮或弹窗外部关闭对话框，不执行操作

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 |
|---------|---------|---------|
| 状态 | 规格组状态（启用/禁用） | TINYINT |

#### 界面元素
- 启用/禁用按钮
- 确认对话框

---

### 3.7 删除规格组

#### 功能说明
删除指定规格组，删除前需要校验是否有关联数据

#### 交互逻辑
1. 在规格组列表的操作按钮中点击"删除"按钮
2. 弹出确认删除对话框
3. 系统执行删除前校验：
   - **校验1：检查是否有规格值**
     - 如果有规格值，对话框显示警告信息："该规格组下有 {数量} 个规格值，删除规格组将同时删除所有规格值，确定要删除吗？"
     - 显示"确认"和"取消"按钮
   - **校验2：检查是否被商品使用**
     - 如果被商品使用，对话框显示警告信息："该规格组被 {数量} 个商品使用，请先取消商品关联"
     - "确认"按钮置灰，不可点击
     - 显示"取消"按钮
   - **校验3：无关联数据**
     - 对话框显示确认信息："确定要删除规格组「{规格组名称}」吗？删除后不可恢复。"
     - 显示"确认"和"取消"按钮
4. 点击"确认"按钮：
   - 调用后端接口删除规格组
   - 同时删除该规格组下的所有规格值
   - 删除成功后关闭对话框，刷新规格组列表
   - 弹出成功提示："规格组删除成功"
5. 点击"取消"按钮或弹窗外部关闭对话框，不执行删除

#### 字段说明
无额外字段，主要是校验逻辑

#### 界面元素
- 确认删除对话框
- 警告信息展示区域
- 确认按钮（校验通过时可用）
- 取消按钮

### 3.8 商品SKU管理（创建商品时使用）

#### 功能说明
在商品创建或编辑时，选择规格属性组合生成SKU

#### 交互逻辑
1. 在商品SPU编辑页面，找到"规格设置"区域
2. **选择规格组**：
   - 点击"添加规格"按钮
   - 弹出规格组选择弹窗
   - 显示所有启用的规格组（可多选）
   - 选择需要的规格组（如颜色、尺寸）
   - 点击"确定"按钮添加
3. **选择规格值**：
   - 每个规格组下方显示该规格组的所有启用规格值
   - 勾选需要的规格值（至少选择一个）
   - 如颜色选择：红色、蓝色、绿色
   - 如尺寸选择：大号、中号、小号
4. **预览SKU**：
   - 点击"预览SKU"按钮
   - 系统根据选中的规格值的笛卡尔积生成SKU列表
   - 显示所有可能的规格组合
   - 如颜色有3个值，尺寸有3个值，生成3×3=9个SKU
5. **批量生成SKU**：
   - 在预览列表中勾选需要生成的SKU（默认全选）
   - 点击"批量生成SKU"按钮
   - 系统自动生成SKU
   - SKU编码和名称自动生成
6. **编辑SKU**：
   - 在生成的SKU列表中可以编辑每个SKU的信息
   - 可编辑字段：
     - SKU编码：可手动修改
     - SKU名称：可手动修改
     - 销售价格：必填，默认为SPU价格
     - 成本价格：可选
     - 库存数量：必填，默认为0
     - 商品图片：可选，可上传SKU专属图片
   - 点击"保存"按钮保存修改
7. **删除SKU**：
   - 点击SKU操作列的"删除"按钮
   - 弹出确认对话框
   - 校验是否有库存或订单
   - 有库存或订单的SKU不能删除
   - 点击"确认"按钮删除
8. **规格组合唯一性校验**：
   - 保存时校验所有SKU的规格组合是否唯一
   - 如果有重复的规格组合，给出错误提示："SKU规格组合不能重复"
   - 高亮显示重复的SKU

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 约束 |
|---------|---------|---------|------|
| SKU编码 | SKU的唯一编码 | VARCHAR(50) | 必填，全局唯一 |
| SKU名称 | SKU的名称 | VARCHAR(200) | 必填 |
| 销售价格 | SKU的销售价格 | DECIMAL(18,2) | 必填，大于0 |
| 成本价格 | SKU的成本价格 | DECIMAL(18,2) | 可选，大于0 |
| 库存数量 | SKU的库存数量 | INT | 必填，大于等于0 |
| 商品图片 | SKU的图片URL | VARCHAR(255) | 可选 |
| 规格组合 | SKU的规格组合（JSON格式） | JSON | 必填，唯一 |

#### 界面元素
- 规格设置区域
- 添加规格按钮
- 规格组选择弹窗
- 规格值勾选区域
- 预览SKU按钮
- SKU预览列表
- 批量生成SKU按钮
- SKU列表（生成后）
- 编辑SKU表单
- 删除SKU按钮

## 四、数据校验规则

### 4.1 规格组名称校验
- 必填校验：规格组名称不能为空
- 长度校验：最多20个字符
- 唯一性校验：规格组名称在系统中唯一
- 特殊字符校验：不能包含特殊字符（支持中文、英文、数字、下划线、横线）

### 4.2 规格组编码校验
- 必填校验：规格组编码不能为空
- 长度校验：最多50个字符
- 格式校验：只能包含英文、数字、下划线
- 唯一性校验：规格组编码在系统中唯一

### 4.3 规格值名称校验
- 必填校验：规格值名称不能为空
- 长度校验：最多20个字符
- 唯一性校验：同一规格组下的规格值名称唯一
- 特殊字符校验：不能包含特殊字符（支持中文、英文、数字、下划线、横线）

### 4.4 规格值编码校验
- 必填校验：规格值编码不能为空
- 长度校验：最多50个字符
- 格式校验：只能包含英文、数字、下划线
- 唯一性校验：规格值编码在系统中唯一

### 4.5 SKU规格组合校验
- 唯一性校验：同一商品SPU下的SKU规格组合必须唯一
- 完整性校验：SKU必须包含所有必填规格组的规格值

### 4.6 删除校验
- 规格组删除校验：有规格值或被商品使用的规格组不能删除
- 规格值删除校验：被SKU使用的规格值不能删除
- SKU删除校验：有库存或订单的SKU不能删除

## 九、总结

本规格属性功能详细设计完整描述了规格属性管理的所有功能点，包括：

1. **规格组列表展示** - 以列表形式展示规格组，支持搜索和筛选

9. **SKU批量操作** - 对SKU进行批量编辑价格、调整库存等操作

每个功能点都包含了详细的功能说明、交互逻辑、字段说明和界面元素，同时明确了业务规则、数据校验规则、异常处理和权限控制，为规格属性功能的开发提供了完整的设计指导。
