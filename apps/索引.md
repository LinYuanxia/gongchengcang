# 应用代码索引（apps 目录）

> 代码结构 ↔ PRD 文档的双向映射索引。
> 开发者导航用：知道代码文件 → 找对应文档 / 知道文档需求 → 找对应代码。

---

## 一、项目总览

```
apps/                     ← 应用代码目录
├── pc/                   ← PC 管理后台（Vue3 + Arco Design）
│   └── src/views/        ← 15 个功能模块
│       ├── construction/ ← 施工方管理
│       ├── dashboard/    ← 仪表盘
│       ├── finance/      ← 财务管理（12 个子模块）
│       ├── login/        ← 登录
│       └── ...           ← 其他模块
│
├── mp/                   ← 微信小程序（UniApp + Vue3）
│   └── src/pages/        ← 8 个页面
│       ├── cart/         ← 购物车
│       ├── index/        ← 首页
│       ├── login/        ← 登录
│       ├── market/       ← 商品市场
│       ├── mine/         ← 我的（18 个页面）
│       ├── order/        ← 订单
│       ├── product/      ← 商品
│       └── warehouse/    ← 仓库
│
packages/                 ← 共享包
├── api/                  ← API 接口层
├── constants/            ← 常量定义
├── types/                ← TypeScript 类型
└── utils/                ← 工具函数
```

## 八、使用指南

| 场景 | 做法 |
|:-----|:-----|
| 找到代码文件，想知道对应的 PRD 需求 | 查 **章节二/三**（代码 → PRD） |
| 在看 PRD 文档，想找到代码实现 | 查 **章节五**（PRD → 代码） |
| 想知道某个模块有哪些子功能 | 查 **章节六/七**（模块速览） |
| 想知道某个共享包被哪些端使用 | 查 **章节四**（共享包索引） |
| 想了解整个项目的应用架构 | 看 **章节一**（项目总览） |
