## 🎯 本期核心范围

### 一、核心业务闭环

| 模块 | 功能点数量 | 说明 |
|:-----|:---------:|------|
| 商品中心 | 8个 | SPU/SKU/分类/属性/供货/上架 |
| 采购链路 | 5个 | 采购计划/采购订单/收货/入库/结算 |
| 销售链路 | 5个 | 商品市场/购物车/订单/出库/对账 |
| 库存管理 | 4个 | 实时库存/库存预警/盘点/调拨 |
| 财务中心 | 6个 | 资金托管/分账/发票/报表/结算 |

### 二、四端功能全景

| 端 | 核心模块 | 说明 |
|:--|:---------|------|
| **工程仓端** | 商品采购 + 库存管理 | 从供应商采购商品，仓库管理 |
| **平台端** | 全局管控 + 运营后台 | 商户/商品/订单/财务/系统全局管理 |
| **供应商端** | 商品供货 + 订单履约 | 维护商品SKU，接收订单发货 |
| **施工方端** | 材料采购 + 工地管理 | BOM清单，工地采购流程 |

> 💡 **版本期数命名规范**：`YY.M.N-关键词`
> - `YY` = 年份后两位（如：26 = 2026）
> - `M` = 月份（如：4 = 4月）
> - `N` = 本月第N个需求
> - `关键词` = 该期需求核心主题
