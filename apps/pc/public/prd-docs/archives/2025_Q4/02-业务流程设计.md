## 一、双交易链路概述

### 1.1 两条交易链路

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          双交易链路架构                                   │
└─────────────────────────────────────────────────────────────────────────┘

链路一：工程仓采购链路（B2B采购）
═══════════════════════════════════════════════════════════════════════════

    ┌──────────┐                    ┌──────────┐
    │  工程仓   │ ─────采购订单────▶ │  供应商   │
    │(Warehouse)│                    │(Supplier)│
    │           │ ◀─────发货─────────│          │
    │           │                    │          │
    │   买方    │                    │   卖方   │
    └──────────┘                    └──────────┘


链路二：施工方采购链路（内部销售）
═══════════════════════════════════════════════════════════════════════════

    ┌──────────┐                    ┌──────────┐
    │  施工方   │ ─────采购订单────▶ │  工程仓   │
    │(Construction)│                 │(Warehouse)│
    │           │ ◀─────发货─────────│          │
    │           │                    │          │
    │   买方    │                    │   卖方   │
    └──────────┘                    └──────────┘


完整链路图：
═══════════════════════════════════════════════════════════════════════════

    ┌──────────┐     采购/供货      ┌──────────┐     采购/销售      ┌──────────┐
    │  供应商   │ ◀─────────────────│  工程仓   │ ◀─────────────────│  施工方   │
    │(Supplier)│                    │(Warehouse)│                   │(Construction)│
    │          │ ─────发货─────────▶│          │ ─────发货────────▶│          │
    └──────────┘                    └──────────┘                   └──────────┘
         │                               │                              │
         │                               │                              │
         └───────────────────────────────┴──────────────────────────────┘
                                    │
                                    ▼
                            ┌──────────────┐
                            │   平台托管    │
                            │  资金结算     │
                            └──────────────┘
```

### 1.2 交易链路对比

| 对比项 | 链路一：工程仓采购 | 链路二：施工方采购 |
|-------|------------------|------------------|
| **买方** | 工程仓 | 施工方 |
| **卖方** | 供应商 | 工程仓 |
| **交易性质** | B2B外部采购 | 内部销售/项目配送 |
| **订单类型** | 采购订单 | 销售订单（项目订单） |
| **资金流向** | 工程仓 → 供应商 | 施工方 → 工程仓 |
| **发票类型** | 供应商开票给工程仓 | 工程仓开票给施工方 |
| **平台服务费** | 向供应商收取 | 向工程仓收取 |
| **库存变化** | 工程仓入库 | 工程仓出库 → 施工方入库 |

|---------|---------|---------|---------|---------|
| BP-001 | 商户入驻流程 | - | 供应商/工程仓/施工方、平台 | 注册→审核→签约→开通 |

| BP-003 | 工程仓采购流程 | 链路一 | 工程仓、供应商 | 选品→下单→支付→确认 |
| BP-004 | 工程仓订单履约流程 | 链路一 | 供应商、工程仓 | 确认→发货→收货→结算 |
| BP-005 | 施工方采购流程 | 链路二 | 施工方、工程仓 | 选品→下单→支付→确认 |
| BP-006 | 施工方订单履约流程 | 链路二 | 工程仓、施工方 | 确认→发货→收货→结算 |
| BP-007 | 项目管理流程 | 链路二 | 工程仓、施工方 | 创建→分配→执行→完成 |
| BP-008 | 工程仓入库流程 | 链路一 | 工程仓 | 创建→验货→入库→更新 |
| BP-009 | 工程仓出库流程 | 链路二 | 工程仓 | 创建→拣货→出库→更新 |
| BP-010 | 施工方库存管理流程 | 链路二 | 施工方 | 入库→领料→退料→报废 |

| BP-018 | BOM包流程 | 双链路 | 工程仓、施工方、供应商 | 创建→发布→采购→拆单 |
| BP-019 | 员工管理流程 | - | 商户管理员 | 新增→分配→激活→管理 |
