## 一、功能概述

### 1.1 功能定位

SKU（Stock Keeping Unit，库存量单位）管理是商品管理的核心功能之一，用于管理商品的具体规格型号。每个SKU对应一个唯一的商品规格，包括规格属性、价格、库存等信息。SKU管理支持新增、编辑、查看、删除SKU，为采购、销售、库存等业务提供基础数据支持。

### 1.2 核心概念

| 概念 | 说明 | 示例 |
|-----|------|------|
| SKU | 库存量单位，对应具体商品规格 | SKU001 |
| SKU编码 | SKU的唯一编码 | SKU202401001 |
| 规格属性 | 商品的规格特征，如颜色、尺寸等 | 颜色:红色, 尺寸:XL |
| 主图 | SKU的主要展示图片 | - |
| 商品相册 | SKU的多张展示图片 | - |
| 供货价 | 供应商提供的价格 | 420元/袋 |
| 销售价 | 面向客户的销售价格 | 450元/袋 |

### 1.3 目标用户

- **平台管理员**：管理所有SKU信息
- **平台运营**：维护SKU信息
- **供应商**：查看自己供货的SKU信息
- **工程仓管理员**：查看SKU信息用于采购

## 二、业务规则

### 2.1 SKU基本信息规则

- **SKU编码**：SKU编码在系统中唯一
  - 格式：SKU + 年月日 + 序号
  - 如"SKU202401001"
  - 系统自动生成或手动输入
- **SKU名称**：SKU名称必填，最多支持200个字符
  - 如"普通硅酸盐水泥P.O42.5 50kg/袋"
- **所属SPU**：每个SKU必须归属一个SPU
  - 从SPU列表中选择
  - 选择后自动填充SPU的单位
- **计量单位**：计量单位必填
  - 如"吨"、"米"、"个"、"件"等
  - 支持手动输入和预设选项
- **条形码**：条形码可选
  - 支持EAN-13、UPC等格式
  - 如"6901234567890"

### 2.2 规格属性规则

- **规格属性组合**：每个SKU的规格属性组合必须唯一
  - 同一SPU下不能有重复的规格组合
  - 如"颜色:红色, 尺寸:XL"和"颜色:红色, 尺寸:L"是不同的SKU
- **规格属性值**：规格属性值必须从SPU绑定的属性中选择
  - 不能自定义规格属性值
  - 如颜色属性只能选择红色、蓝色、绿色等值
- **规格展示**：规格属性以标签形式展示
  - 每个规格属性显示为一个标签
  - 如"颜色:红色"、"尺寸:XL"

### 2.3 图片上传规则

- **主图**：主图可选，最多上传1张
  - 支持JPG、PNG格式
  - 建议尺寸：800x800像素
  - 文件大小不超过5MB
- **商品相册**：商品相册可选，最多上传9张
  - 支持JPG、PNG格式
  - 建议尺寸：800x800像素
  - 文件大小不超过5MB
- **图片预览**：支持图片预览功能
  - 点击图片可放大查看
  - 支持左右切换查看多张图片

### 2.4 价格信息规则

- **供货价**：供货价可选，必须大于等于0
  - 保留2位小数
  - 如"420.00元/袋"
- **销售价**：销售价可选，必须大于等于0
  - 保留2位小数
  - 如"450.00元/袋"
- **价格校验**：销售价建议大于等于供货价
  - 如销售价低于供货价，给出警告提示
  - 但允许保存

### 2.5 状态管理规则

- **上架状态**：SKU状态分为上架和下架
  - 上架：正常使用，可参与采购、销售
  - 下架：暂停使用，不可参与采购、销售
- **状态变更**：支持手动切换状态
  - 上架→下架：确认后下架
  - 下架→上架：确认后上架
- **状态影响**：下架的SKU在选择时不显示
  - 采购、销售等业务只能选择上架的SKU
  - 但可以查看历史数据中的下架SKU

### 2.6 删除约束规则

- **SKU删除**：删除SKU前必须检查是否有关联数据
  - 检查是否有采购订单：有采购订单的SKU不能直接删除
  - 检查是否有销售订单：有销售订单的SKU不能直接删除
  - 检查是否有库存记录：有库存记录的SKU不能直接删除
  - 检查是否有供货关系：有供应商供货的SKU不能直接删除
- **删除提示**：删除时根据校验结果给出明确的提示信息
  - 有采购订单时提示："该SKU下有采购订单，不能删除"
  - 有供货关系时提示："该SKU下有供应商供货，不能删除"

### 2.7 SKU列表展示规则

- **默认排序**：SKU列表默认按创建时间倒序展示
  - 最新创建的SKU显示在最前面
  - 支持按SKU编码、SKU名称、创建时间进行排序切换
- **分页展示**：当SKU数量超过10条时采用分页展示
  - 每页显示10条SKU
  - 支持跳转到指定页码
  - 显示总记录数和当前页码信息
- **搜索规则**：搜索功能支持模糊匹配
  - 按SKU名称搜索：支持包含匹配（如输入"水泥"可匹配"普通硅酸盐水泥"）
  - 按SKU编码搜索：支持前缀匹配（如输入"SKU2024"可匹配SKU202401001）
- **筛选规则**：支持按商品分类和SPU筛选SKU
  - 按商品分类筛选：支持多级分类筛选
  - 按SPU筛选：支持从SPU列表跳转过来时自动筛选

## 三、功能点详细设计

### 3.1 SKU列表展示

#### 功能说明
以列表形式展示SKU信息，支持搜索、筛选、分页加载

#### 交互逻辑
1. 页面加载时显示所有SKU，默认按创建时间倒序排列
2. 支持按SKU名称/编码进行搜索
3. 支持按商品分类筛选SKU
4. 支持按SPU筛选SKU（从SPU列表跳转过来时自动筛选）
5. SKU列表展示字段：
   - 主图：SKU主图，点击可放大查看
   - SKU编码：SKU唯一编码
   - SKU名称：SKU名称
   - 所属SPU：所属SPU名称
   - 所属分类：所属分类名称
   - 规格：规格属性组合，以标签形式展示
   - 单位：计量单位
   - 供应商数：供货的供应商数量
   - 创建时间：SKU创建时间
   - 操作按钮：详情、编辑、删除
6. 支持分页加载，当SKU数量较多时采用分页方式

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 展示方式 |
|---------|---------|---------|---------|
| 主图 | SKU主图 | VARCHAR | 图片展示，点击可放大 |
| SKU编码 | SKU唯一编码 | VARCHAR | 文本展示 |
| SKU名称 | SKU名称 | VARCHAR | 文本展示 |
| 所属SPU | 所属SPU名称 | VARCHAR | 文本展示 |
| 所属分类 | 所属分类名称 | VARCHAR | 文本展示 |
| 规格 | 规格属性组合 | JSON | 标签展示，每个规格一个标签 |
| 单位 | 计量单位 | VARCHAR | 文本展示 |
| 供应商数 | 供货的供应商数量 | INT | 文本展示 |
| 创建时间 | SKU创建时间 | DATETIME | 文本展示，格式YYYY-MM-DD HH:mm:ss |

#### 界面元素
- 搜索区域：
  - SKU名称/编码搜索框：支持模糊搜索
  - 商品分类级联选择器：支持多级分类筛选
- 列表区域：
  - 表格展示所有SKU
  - 支持分页
- 操作按钮：
  - 详情：查看SKU详情
  - 编辑：编辑SKU信息
  - 删除：删除SKU信息
- 筛选提示：
  - 当从SPU列表跳转过来时，显示SPU名称和清除筛选按钮

#### 权限说明
- 平台管理员：可查看所有SKU
- 平台运营：可查看所有SKU
- 供应商：只能查看自己供货的SKU
- 工程仓管理员：可查看所有上架的SKU

#### 异常处理
- 网络异常：提示"网络异常，请稍后重试"
- 无数据：提示"暂无数据"
- 搜索无结果：提示"未找到符合条件的SKU"

### 3.2 新增SKU

#### 功能说明
创建新的SKU信息

#### 交互逻辑
1. 点击"新增SKU"按钮，弹出新增SKU抽屉
2. 填写基本信息：
   - 所属SPU：选择所属SPU，必填
   - SKU编码：输入SKU编码，必填
   - SKU名称：输入SKU名称，必填
   - 规格属性：根据SPU的规格属性选择，必填
   - 单位：输入单位，必填
   - 条形码：输入条形码，选填
   - 主图：上传主图，选填
3. 点击"确定"按钮，校验表单完整性
4. 校验通过后保存SKU信息
5. 保存成功后关闭抽屉，刷新SKU列表，弹出成功提示："新增成功"
6. 点击"取消"按钮或抽屉外部，关闭抽屉，不保存数据

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 必填 | 验证规则 |
|---------|---------|---------|------|---------|
| 所属SPU | 所属SPU ID | VARCHAR | 是 | - |
| SKU编码 | SKU编码 | VARCHAR | 是 | 1-50个字符 |
| SKU名称 | SKU名称 | VARCHAR | 是 | 1-200个字符 |
| 规格属性 | 规格属性组合 | JSON | 是 | 必须选择所有规格属性 |
| 单位 | 计量单位 | VARCHAR | 是 | 1-20个字符 |
| 条形码 | 条形码 | VARCHAR | 否 | 最多50个字符 |
| 主图 | 主图URL | VARCHAR | 否 | 最多500个字符 |

#### 界面元素
- 抽屉标题：新增SKU
- 基本信息表单：
  - 所属SPU选择器：支持搜索
  - SKU编码输入框
  - SKU名称输入框
  - 规格属性选择器：根据SPU动态生成
  - 单位输入框
  - 条形码输入框
  - 主图上传组件
- 底部操作栏：
  - 取消按钮
  - 确定按钮

#### 权限说明
- 平台管理员：可新增SKU
- 平台运营：可新增SKU

#### 异常处理
- 表单验证失败：提示具体错误信息
- SKU编码重复：提示"SKU编码已存在"
- 规格组合重复：提示"该规格组合已存在"
- 保存失败：提示"新增SKU失败，请稍后重试"

### 3.3 编辑SKU

#### 功能说明
编辑已有的SKU信息

#### 交互逻辑
1. 点击"编辑"按钮，弹出编辑SKU抽屉
2. 显示当前SKU的信息
3. 可修改的字段：
   - SKU名称：可修改
   - 规格属性：可修改
   - 单位：可修改
   - 条形码：可修改
   - 主图：可修改
   - 所属SPU：不可修改
   - SKU编码：不可修改
4. 点击"确定"按钮，校验表单完整性
5. 校验通过后更新SKU信息
6. 更新成功后关闭抽屉，刷新SKU列表，弹出成功提示："编辑成功"
7. 点击"取消"按钮或抽屉外部，关闭抽屉，不保存数据

#### 字段说明
同新增SKU

#### 界面元素
同新增SKU

#### 权限说明
- 平台管理员：可编辑SKU
- 平台运营：可编辑SKU

#### 异常处理
- SKU不存在：提示"SKU不存在"
- 表单验证失败：提示具体错误信息
- 规格组合重复：提示"该规格组合已存在"
- 更新失败：提示"编辑SKU失败，请稍后重试"

### 3.4 查看SKU详情

#### 功能说明
查看SKU的详细信息，包括基本信息、价格信息、分账配置、供应商信息、库存记录、操作记录

#### 交互逻辑
1. 在SKU列表的操作按钮中点击"详情"按钮
2. 跳转到SKU详情页面
3. **基本信息**：
   - SKU编码
   - SKU名称
   - 所属SPU：可点击跳转到SPU详情
   - 所属分类
   - 计量单位
   - 状态：上架/下架
   - 条形码
   - 供应商数
   - 规格组合：以标签形式展示
   - SKU主图：显示主图，点击可放大
   - 商品相册：显示所有相册图片，支持左右切换
4. **价格信息**：
   - 供货价
   - 销售价
5. **分账配置**：
   - 显示分账规则列表
   - 规则名称、规则类型、分账比例、生效时间、状态
   - 操作按钮：查看
6. **材料供应商**：
   - 显示供货的供应商列表
   - 供应商名称、供货价、预计供货量、最小起订量、供货周期、状态
   - 操作按钮：查看全部供应商商品
7. **库存记录**：
   - 显示库存记录列表
   - 暂无数据时显示"暂无库存记录"
8. **操作记录**：
   - 显示操作记录列表
   - 暂无数据时显示"暂无操作记录"
9. 点击"查看SPU"按钮：跳转到SPU详情页面
10. 点击"编辑"按钮：跳转到SKU编辑页面
11. 点击"返回"按钮：返回到SKU列表页面

#### 字段说明

| 所属分类 | 所属分类名称 | VARCHAR | 文本展示 |
| 计量单位 | 计量单位 | VARCHAR | 文本展示 |
| 状态 | SKU状态 | VARCHAR | 标签展示，上架（绿色）/下架（红色） |
| 条形码 | 条形码 | VARCHAR | 文本展示 |
| 供应商数 | 供货的供应商数量 | INT | 文本展示 |
| 规格组合 | 规格属性组合 | JSON | 标签展示 |
| SKU主图 | SKU主图 | VARCHAR | 图片展示，点击可放大 |
| 商品相册 | 商品相册图片 | ARRAY | 图片展示，支持左右切换 |
| 供货价 | 供货价 | DECIMAL | 文本展示，保留2位小数 |
| 销售价 | 销售价 | DECIMAL | 文本展示，保留2位小数 |

#### 界面元素

- 操作按钮：查看SPU、编辑

  - 价格信息描述列表

  - 分账规则表格
- 材料供应商卡片：
  - 供应商表格

- 库存记录卡片：
  - 库存记录表格
- 操作记录卡片：
  - 操作记录表格

#### 权限说明

- 文件数量超出限制：提示"最多上传9张图片"
- 上传失败：提示"图片上传失败，请稍后重试"

### 3.6 搜索SKU

#### 功能说明
在搜索框输入关键词，支持模糊搜索SKU

#### 交互逻辑
1. 搜索框位于页面顶部，默认显示占位文字"搜索SKU名称/编码"
2. 输入关键词后点击搜索按钮触发搜索
3. 支持按回车键触发搜索
4. 搜索结果处理：
   - 匹配的SKU在列表中展示
   - 匹配的关键词高亮显示
   - 无匹配结果时显示"暂无搜索结果"提示
5. 清空搜索框时恢复SKU列表的原始展示状态

#### 字段说明
| 字段名称 | 字段说明 | 匹配规则 |
|---------|---------|---------|
| SKU名称 | 按SKU名称模糊匹配 | 包含关键词即匹配 |
| SKU编码 | 按SKU编码精确匹配或前缀匹配 | 完全匹配或前缀匹配 |

#### 界面元素
- 搜索输入框
- 搜索图标
- 搜索结果提示（无结果时显示）

#### 权限说明
- 所有用户：可搜索SKU

#### 异常处理
- 搜索失败：提示"搜索失败，请稍后重试"

### 3.7 筛选SKU

#### 功能说明
按商品分类或SPU筛选SKU

#### 交互逻辑
1. **按商品分类筛选**：
   - 点击商品分类级联选择器
   - 选择分类（支持多级分类）
   - 选择后自动筛选该分类下的SKU
   - 支持清除筛选条件
2. **按SPU筛选**：
   - 从SPU列表点击SKU数量跳转到SKU列表
   - 自动筛选该SPU下的SKU
   - 显示SPU名称和清除筛选按钮
   - 点击清除筛选按钮恢复所有SKU

#### 字段说明
| 字段名称 | 字段说明 |
|---------|---------|
| 商品分类 | 按商品分类筛选 |
| SPU | 按SPU筛选 |

#### 界面元素
- 商品分类级联选择器
- SPU筛选提示和清除按钮

#### 权限说明
- 所有用户：可筛选SKU

#### 异常处理
- 筛选失败：提示"筛选失败，请稍后重试"

### 3.8 删除SKU

#### 功能说明
删除已有的SKU信息

#### 交互逻辑
1. 在SKU列表的操作按钮中点击"删除"按钮
2. 弹出删除确认对话框，提示："确定要删除该SKU吗？删除后将无法恢复。"
3. 点击"确定"按钮，系统进行删除前校验：
   - 检查是否有采购订单：有采购订单则提示"该SKU下有采购订单，不能删除"
   - 检查是否有销售订单：有销售订单则提示"该SKU下有销售订单，不能删除"
   - 检查是否有库存记录：有库存记录则提示"该SKU下有库存记录，不能删除"
   - 检查是否有供货关系：有供货关系则提示"该SKU下有供应商供货，不能删除"
4. 校验通过后执行删除操作
5. 删除成功后关闭对话框，刷新SKU列表，弹出成功提示："删除成功"
6. 点击"取消"按钮或对话框外部，关闭对话框，不执行删除操作

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 必填 |
|---------|---------|---------|------|
| skuId | SKU ID | VARCHAR | 是 |

#### 界面元素
- 删除按钮：位于SKU列表操作列
- 删除确认对话框：
  - 提示文本："确定要删除该SKU吗？删除后将无法恢复。"
  - 取消按钮
  - 确定按钮（红色）

#### 权限说明
- 平台管理员：可删除SKU
- 平台运营：可删除SKU

#### 异常处理
- SKU不存在：提示"SKU不存在"
- 有采购订单：提示"该SKU下有采购订单，不能删除"
- 有销售订单：提示"该SKU下有销售订单，不能删除"
- 有库存记录：提示"该SKU下有库存记录，不能删除"
- 有供货关系：提示"该SKU下有供应商供货，不能删除"
- 删除失败：提示"删除SKU失败，请稍后重试"
