# 供应商端 - 系统操作手册

> 版本：v2.0 | 更新日期：2026-04-27 | 适用角色：管理员、业务员、仓管员、财务人员、客服

---

## ⚠️ 操作前必读

| 规则 | 说明 | 后果 |
|:----|:----|:----|
| 订单确认后不可取消 | 确认接单即进入备货流程，无法撤销 | 误确认需联系工程仓协商，无法系统取消 |
| 商品下架需审核 | 下架后重新上架必须提交平台审核 | 上架期间商品不可售，影响销售 |
| 结算单不可修改 | 结算单按自然月生成后无法修改 | 请每月底核对订单数据，有误及时联系平台 |
| 发票不可重复关联 | 一笔订单只能关联一个发票 | 重复关联会被系统拦截提示 |
| 操作前核实数据 | 发货/结算等重要操作前务必核实信息 | 数据错误需人工处理，影响效率 |

## 📖 二、快速入门（5分钟上手）

### 2.1 登录系统

> 📸 **[截图占位符]**：登录页，标注要点：①账号输入框 ②密码输入框 ③登录按钮 ④忘记密码入口

1. 打开浏览器，访问管理员提供的供应商端系统地址
2. 输入管理员分配的 **账号**（通常是手机号）和 **密码**
3. 点击 **「登录」** 按钮
4. 登录成功后，默认进入 **工作台** 页面

> 💡 **提示**：如果忘记密码，请联系管理员重置。连续5次输错密码，账号会被临时锁定。

> 📸 **[截图占位符]**：登录成功后进入的工作台首页，标注要点：①时间筛选 ②指标卡片 ③快捷入口

### 2.2 完成第一个任务：处理一笔采购订单

以最常见的"确认订单→发货"为例：

> 📸 **[截图占位符]**：订单管理-待确认Tab，标注要点：①Tab切换区 ②待确认订单列表 ③确认按钮 ④取消按钮

1. 在左侧菜单点击 **「订单管理」**
2. 默认展示"待确认"Tab，列表中显示工程仓发来的采购订单
3. 找到您要处理的订单，点击右侧 **「确认」** 按钮
   > ⚠️ **重要**：确认接单后无法取消，请核实订单信息无误后再确认
4. 订单状态变为"待发货"

> 📸 **[截图占位符]**：订单管理-待发货Tab + 发货弹窗，标注要点：①待发货列表 ②发货按钮 ③物流公司下拉 ④物流单号输入 ⑤提交按钮

5. 切换到"待发货"Tab，找到刚确认的订单，点击 **「发货」**
6. 填写物流信息（物流公司/单号可选填），点击 **「提交」**
7. 订单状态变为"已发货"，等待工程仓确认收货

## 🧭 二、功能详解

### 2.1 工作台与商户中心

#### 2.1.1 数据概览（工作台）

> 📸 **[截图占位符]**：工作台首页，标注要点：①时间筛选（今日/本周/本月）②5个指标卡片 ③快捷入口

**页面概览**

工作台是登录后的首页，展示日常运营的核心数据：

| 标注 | 名称 | 作用 |
|:---|:---|:---|
| ① | 时间筛选 | 切换"今日/本周/本月"查看不同时段数据 |
| ② | 指标卡片 | 展示订单总金额、订单数、待发货、待结算金额、在售商品数 |
| ③ | 快捷入口 | 点击指标卡片可快速跳转到对应模块 |

**操作步骤**

**如何查看运营数据**
1. 登录后自动进入工作台
2. 默认展示"今日"数据
3. 点击 **「本周」** 或 **「本月」** 可切换时间范围
4. 点击任意指标卡片（如"待发货"），直接跳转到订单管理相关Tab

> 💡 **提示**：指标卡片加载失败时会显示"--"，稍后刷新即可恢复。

#### 2.1.2 主体信息

**如何查看和编辑企业信息**
1. 左侧菜单点击 **「商户信息」→「主体信息」**
2. 查看企业工商信息（公司全称、统一信用代码、法人代表等）
3. 如您有管理员权限，可点击 **「编辑」** 按钮修改联系人姓名、手机号、邮箱
4. 修改后点击 **「保存」**，提示"修改成功"

> ⚠️ **注意**：公司全称、信用代码等工商信息由平台审核录入，供应商端不可修改。

#### 2.1.3 合同列表

1. 左侧菜单点击 **「商户信息」→「合同列表」**
2. 查看供应商与平台签署的合作协议
3. 点击某条合同可查看详情并下载附件

### 2.2 商品中心

#### 页面概览

> 📸 **[截图占位符]**：商品中心列表页，标注要点：①Tab切换（在售/待审核/已下架）②搜索区 ③商品列表 ④行操作按钮

| 标注 | 名称 | 作用 |
|:---|:---|:---|
| ① | Tab切换 | 按商品状态筛选：在售/待审核/已下架/审核不通过 |
| ② | 搜索区 | 按商品名称/SKU编码搜索 |
| ③ | 商品列表 | 展示商品图片、名称、SKU、供货价、库存、审核状态 |
| ④ | 行操作 | 编辑、上下架等操作按钮 |

#### 操作步骤

**如何新增商品**

> 📸 **[截图占位符]**：新增商品弹窗，标注要点：①商品搜索 ②选择结果 ③供货价输入 ④库存输入 ⑤提交按钮

1. 左侧菜单点击 **「商品中心」**
2. 点击 **「新增商品」** 按钮
3. 在弹出窗口中，从平台商品库搜索并选择要供货的商品
4. 填写 **供货价**（必须大于0）和 **库存**（必须≥0）
5. 点击 **「提交」**，商品进入"待审核"状态
6. 等待平台审核通过后，商品变为"在售"状态

> ⚠️ **注意**：
> - 供货价必须大于0元
> - 如果提示"该商品已添加"，表示该商品之前已经添加过了
> - 提交后无法自行修改供货价，如需修改请等审核结果

**如何上下架商品**

> 📸 **[截图占位符]**：下架确认弹窗，标注要点：①提示文案 ②确认按钮 ③取消按钮

1. 在商品列表中找到需要操作的商品
2. 点击 **「下架」** → 确认弹窗中点击 **「确认」**
   > ⚠️ **注意**：下架后工程仓端立即不可见，重新上架需提交平台审核
3. 商品变为"已下架"状态，工程仓不再可见
4. 如需重新上架，点击 **「上架」** 提交平台审核

**如何查看库存**
1. 在商品列表中，库存列直接显示当前库存数量
2. 点击库存数字，可查看该商品的 **库存流水**（每一次库存变动的记录）

> 📸 **[截图占位符]**：库存流水弹窗，标注要点：①商品信息 ②流水列表（时间/类型/数量/备注）

#### 字段说明

| 字段名称 | 说明 | 示例 |
|:---|:---|:---|
| 商品名称 | 商品的名称 | 325水泥（袋装） |
| SKU编码 | 系统自动生成的唯一编码 | SP20240001 |
| 规格 | 商品的规格属性 | 50kg/袋 |
| 供货价 | 供应商设定的销售单价 | ¥300.00 |
| 库存 | 当前库存数量 | 500 |
| 审核状态 | 商品当前的状态（在售/待审核/已下架/审核不通过） | **在售** 🟢 |

#### 状态说明

| 状态 | 颜色 | 含义 | 可执行操作 |
|:---|:---|:---|:---|
| 在售 | 🟢 | 审核通过，工程仓可采购 | 下架、编辑 |
| 待审核 | 🟡 | 已提交等待平台审核 | 编辑 |
| 已下架 | ⚪ | 主动下架，工程仓不可见 | 上架、编辑 |
| 审核不通过 | 🔴 | 平台驳回，需修改后重提 | 编辑后重新提交 |

#### 状态流转图

```
[新增商品] → 提交 → [待审核 🟡] → 平台审核通过 → [在售 🟢] → 下架 → [已下架 ⚪]
                              ↓                                       ↓
                       审核不通过 🔴                             上架(需审核)
                              ↓                                       ↓
                       [编辑后重提]                              [待审核 🟡]
```

### 2.3 订单管理

#### 页面概览

> 📸 **[截图占位符]**：订单管理列表页，标注要点：①Tab筛选（6个状态Tab）②搜索区 ③订单列表 ④行操作按钮

| 标注 | 名称 | 作用 |
|:---|:---|:---|
| ① | Tab筛选 | 全部/待确认/待发货/已发货/已完成/已取消（含状态数量角标） |
| ② | 搜索区 | 按订单编号/工程仓名称搜索 |
| ③ | 订单列表 | 展示订单编号、工程仓、商品数量、金额、状态、下单时间 |
| ④ | 行操作 | 根据状态动态显示：确认/发货/打印/售后等 |

#### 操作步骤

**如何确认接单**

> 📸 **[截图占位符]**：确认接单弹窗，标注要点：①订单信息摘要 ②确认按钮 ③取消按钮

1. 左侧菜单点击 **「订单管理」**
2. 默认进入"待确认"Tab，查看工程仓发来的采购订单
3. 点击订单右侧的 **「确认」** 按钮（无需二次确认）
   > ⚠️ **重要**：确认接单后即进入备货流程，无法系统取消，请核实订单信息
4. 订单状态变为"待发货"，工程仓会收到通知

> 💡 **提示**：确认接单后即进入备货流程，如果因特殊原因无法供货，请联系工程仓协商处理。

**如何取消订单**
1. 在"待确认"Tab中，找到需要取消的订单
2. 点击 **「取消」** 按钮
3. 确认后订单变为"已取消"状态

> ⚠️ **注意**：仅"待确认"状态下的订单可以取消，已确认的订单无法取消。

**如何发货**

> 📸 **[截图占位符]**：发货弹窗，标注要点：①物流公司下拉选择 ②物流单号输入 ③发货备注 ④提交按钮 ⑤取消按钮

1. 在"待发货"Tab中，找到需要发货的订单
2. 点击 **「发货」** 按钮
3. 在弹窗中填写发货信息：
   - **物流公司**：可选填（支持自有配送）
   - **物流单号**：可选填
   - **发货备注**：可选填（最多200字）
4. 点击 **「提交」**，订单状态变为"已发货"

> 💡 **提示**：支持分批发货，每次发货记录自动汇总到订单详情中。

**如何打印发货单**
1. 在"已发货"Tab或订单详情页中
2. 点击 **「打印」** 按钮
3. 预览发货单（含供应商信息、工程仓信息、商品明细、日期）
4. 点击浏览器打印功能完成打印

**如何处理售后补发**

> 📸 **[截图占位符]**：售后处理弹窗，标注要点：①货损证据展示 ②同意补发按钮 ③拒绝补发+原因输入

1. 在"已完成"Tab中找到需售后的订单
2. 点击 **「售后」** 进入售后详情
3. 查看货损证据
4. 选择操作：
   - **同意补发**：系统自动生成补发订单，进入发货流程
   - **拒绝补发**：必须填写拒绝原因

> 💡 **提示**：售后完成后，原订单状态不变，补发订单独立跟踪。

#### 字段说明

| 字段名称 | 说明 | 示例 |
|:---|:---|:---|
| 订单编号 | 系统生成的唯一订单号 | ORD20240424001 |
| 工程仓名称 | 下单的工程仓企业名称 | XX建材工程仓 |
| 商品数量 | 该订单包含的商品种类和数量 | 2种共100件 |
| 订单金额 | 订单总金额 | ¥30,000.00 |
| 订单状态 | 当前所处的订单生命周期阶段 | **待发货** 🟡 |
| 下单时间 | 工程仓提交订单的时间 | 2026-04-24 14:30 |

#### 状态说明

| 状态 | 颜色 | 含义 | 可执行操作 |
|:---|:---|:---|:---|
| 待确认 | 🟡 | 工程仓已下单，等待供应商接单 | 确认、取消 |
| 待发货 | 🔵 | 已接单，等待发货 | 发货 |
| 已发货 | 🟢 | 已填写物流信息发货成功 | 打印、查看物流 |
| 已完成 | ✅ | 工程仓确认收货 | 查看、售后处理 |
| 已取消 | ⭕ | 订单已作废 | 查看 |

#### 状态流转图

```
[待确认 🟡] → 确认接单 → [待发货 🔵] → 发货 → [已发货 🟢] → 工程仓收货 → [已完成 ✅]
    ↓                                                        ↓
  取消                                                  申请售后
    ↓                                                        ↓
[已取消 ⭕]                                           [售后处理中 → 补发/关闭]
```

> 💡 **提示**：售后处理不影响原订单状态，补发订单走独立新流程。

### 2.4 财务中心

#### 2.4.1 发票管理

> 📸 **[截图占位符]**：发票管理列表页，标注要点：①新增发票按钮 ②关联订单按钮 ③发票列表 ④行操作（下载/关联）

**页面概览**

| 标注 | 名称 | 作用 |
|:---|:---|:---|
| ① | 操作按钮 | 新增发票、关联订单 |
| ② | 发票列表 | 展示发票号码、关联订单、金额、开票日期、状态 |

**如何新增发票**

> 📸 **[截图占位符]**：新增发票弹窗，标注要点：①文件上传区 ②发票号码输入 ③发票金额输入 ④订单选择 ⑤提交按钮

1. 左侧菜单点击 **「财务中心」→「发票管理」**
2. 点击 **「新增发票」** 按钮
3. 上传发票文件（支持PDF/JPG/PNG，不超过10MB）
4. 填写 **发票号码** 和 **发票金额**
5. 选择要关联的订单
6. 点击 **「提交」** 保存发票记录

> 💡 **提示**：一笔订单只能关联一个发票，不能重复关联。如果订单已被关联过，选择列表中不可选。

**如何关联订单**
1. 在发票列表中点击未关联订单的发票
2. 点击 **「关联订单」** 按钮
3. 在弹窗中选择未关联的订单
4. 确认后完成关联

**如何下载发票**
1. 在发票列表中找到需要下载的发票
2. 点击 **「下载」** 按钮即可获取发票文件

#### 2.4.2 待结算与结算单

> 📸 **[截图占位符]**：待结算页面，标注要点：①待结算订单列表 ②结算状态列 ③金额汇总

**如何查看待结算**
1. 左侧菜单点击 **「财务中心」→「待结算」**
2. 查看已发货且工程仓确认收货但未结算的订单列表
3. 字段：订单编号、工程仓、订单金额、已发货日期、结算状态

**如何查看结算单**

> 📸 **[截图占位符]**：结算单列表页，标注要点：①结算单列表 ②结算周期 ③结算金额 ④状态

1. 左侧菜单点击 **「财务中心」→「结算单」**
2. 查看已生成的结算记录
3. 字段：结算单编号、结算周期、结算金额、订单数量、状态

> ⚠️ **注意**：结算单按自然月生成，生成后不可修改。

### 2.5 系统设置

> 📸 **[截图占位符]**：系统设置-账号列表页，标注要点：①账号列表 ②新增按钮 ③编辑按钮 ④禁用/启用开关

#### 2.5.1 账号管理

**如何查看和管理账号**
1. 左侧菜单点击 **「系统设置」→「账号列表」**
2. 列表展示：账号名、所属员工、角色、状态、最后登录时间
3. 点击 **「新增」** 创建新账号
4. 点击 **「编辑」** 修改账号信息或角色
5. 点击 **「禁用/启用」** 开关控制账号的登录权限

> ⚠️ **注意**：禁用账号后，该员工将无法登录系统。请确认该员工已无工作需要再操作。

#### 2.5.2 员工管理

**如何管理员工**
1. 左侧菜单点击 **「系统设置」→「员工管理」**
2. 列表展示：员工姓名、手机号、岗位、状态（在职/离职）
3. 点击 **「新增」** 添加新员工
4. 点击 **「编辑」** 修改员工信息
5. 员工离职后点击 **「标记离职」**

#### 2.5.3 角色管理与权限配置

> 📸 **[截图占位符]**：权限配置弹窗，标注要点：①菜单树勾选 ②操作按钮勾选 ③保存按钮 ④取消按钮

**如何管理角色**
1. 左侧菜单点击 **「系统设置」→「角色列表」**
2. 列表展示：角色名称、编码、拥有该角色的员工数
3. 点击 **「新增」** 创建自定义角色
4. 点击 **「编辑」** 修改角色名称

**如何配置权限**
1. 在角色列表中点击 **「权限配置」**
2. 在弹窗中按菜单+操作按钮粒度勾选权限
3. 勾选完成后点击 **「保存」**
4. 保存后即时生效，对应角色的权限立即更新

> 💡 **提示**：系统预设了5个角色（管理员、业务员、仓管员、财务、客服），可直接使用或自定义修改。

#### 字段说明

| 字段名称 | 说明 | 示例 |

## ❓ 三、常见问题（FAQ）

| # | 问题 | 答案 |
|:-:|:---|:----|
| 1 | 登录时提示"账号或密码错误"怎么办？ | 请检查输入的账号名和密码是否正确。连续5次输错密码账号会被临时锁定，请联系管理员解锁。 |
| 2 | 新增商品时提示"该商品已添加"是什么意思？ | 表示该商品已在您的商品列表中，无需重复添加，直接到列表中查找即可。 |
| 3 | 如何新增商品？ | 进入「商品中心」→ 点击「新增商品」→ 从平台商品库选择商品 → 填写供货价和库存 → 提交等待平台审核。详情参见 2.2 商品中心章节。 |
| 4 | 工程仓下的订单，我怎么知道？ | 当工程仓提交采购订单后，订单会出现在「订单管理」的"待确认"Tab中，登录后即可看到。 |
| 5 | 订单发货时一定要填写物流单号吗？ | 不需要。物流公司和物流单号都是非必填项，如果您自配送，直接提交即可。 |
| 6 | 可以分多次发货吗？ | 可以。支持分批发货，每次发货记录会自动汇总到订单详情中。 |
| 7 | 商品下架了，工程仓那边能看到吗？ | 不能。下架后工程仓端不再展示该商品，不能采购。重新上架需要提交平台审核。 |
| 8 | 发票上传大小和格式有什么要求？ | 支持 PDF、JPG、PNG 格式，文件大小不超过10MB。 |
| 9 | 售后补发处理流程是怎样的？ | 查看货损证据 → 同意补发（自动生成补发订单，走发货流程）/ 拒绝补发（必须填写拒绝原因）。 |
| 10 | 供货价填错了怎么办？ | 如果商品还在"待审核"状态，可以编辑修改。如果已经是"在售"状态，需先下架再重新提交审核。 |
| 11 | 如何给员工分配不同的权限？ | 进入「系统设置」→「角色列表」→ 点击「权限配置」→ 按菜单和按钮粒度勾选权限 → 保存后即生效。然后在「账号管理」中给员工分配对应角色。 |
| 12 | 结算周期是多久？ | 按自然月结算。已发货且工程仓确认收货的订单进入待结算，结算单生成后不可修改。 |
| 13 | 订单确认接单后发现价格不对怎么办？ | 确认接单后无法取消或修改，请第一时间联系工程仓协商，由平台端协调处理。 |
| 14 | 发货时提示"库存不足"为什么？ | 您的商品库存不足以满足发货数量，请先更新商品库存后再操作。 |

## ⚠️ 四、常见错误提示及处理方法

| 提示信息 | 含义 | 处理方法 |
|:--------|:----|:--------|
| "账号或密码错误" | 账号密码不匹配，或账号被锁定 | 检查输入是否正确；如连续输错请找管理员解锁 |
| "该商品已添加" | 该商品已在商品列表中存在 | 无需重复添加，直接到列表中查找编辑 |
| "供货价必须大于0" | 填写的供货价不合法 | 输入大于0的价格，如 300.00 |
| "库存必须≥0" | 库存数量不合法 | 输入大于等于0的整数 |
| "文件大小超过10MB" | 上传的发票文件过大 | 压缩图片或PDF文件后再上传 |
| "库存不足" | 发货数量超过当前库存 | 先补充库存，再重新发货 |
| "订单已被关联" | 该订单已和其他发票关联 | 检查订单关联状态，不可重复关联 |
| "请填写拒绝原因" | 拒绝售后补发时必须说明原因 | 填写具体拒绝原因（必填） |
| "网络请求失败" | 网络连接异常 | 检查网络连接后刷新页面重试 |
| "操作失败，请稍后重试" | 系统临时异常 | 稍后刷新再试，如持续失败请联系技术支持 |

## ℹ️ 五、注意事项

| # | 规则 | 详细说明 |
|:-:|:----|:--------|
| 1 | **供货价设置** | 新增商品时，供货价必须大于0，库存必须≥0 |
| 2 | **订单取消** | 仅"待确认"状态的订单可以取消，确认接单后无法取消 |
| 3 | **发货填单** | 物流公司和非物流单号均非必填，支持自有配送 |
| 4 | **售后补发** | 仅"已完成"订单可发起售后；拒绝补发必须填写原因 |
| 5 | **发票唯一** | 一笔订单只能关联一个发票，不能重复关联 |
| 6 | **权限生效** | 权限配置保存后即时生效，无需重新登录 |
| 7 | **账号禁用** | 禁用账号后，该员工将无法登录系统 |
| 8 | **敏感信息** | 主体信息中的公司全称、信用代码为平台录入，不可修改 |
| 9 | **密码安全** | 建议定期修改密码，不要使用简单密码 |
