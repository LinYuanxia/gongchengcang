## 一、功能概述

### 1.1 功能定位

系统设置是供应商端的**管理配置模块**，提供账号管理、员工管理、角色管理和权限配置能力。

### 1.2 核心概念

| 概念 | 说明 |
|-----|------|
| 账号 | 供应商的登录账号，对应具体员工 |
| 角色 | 权限集合，不同角色拥有不同的操作权限 |
| 权限配置 | 为角色勾选具体的功能操作权限 |

### 1.3 模块范围

| 功能分类 | 主要功能 | 优先级 |
|---------|---------|:------:|
| 账号管理 | 账号列表 | P1 |
| 员工管理 | 员工列表 | P1 |
| 角色管理 | 角色列表、权限配置 | P1 |

---

## 二、业务规则

### 2.1 账号规则

- 一个员工只能有一个账号
- 账号创建后自动以员工手机号为登录名
- 账号状态：启用/禁用

### 2.2 角色规则

- 预设角色：管理员、业务员、仓管员、财务、客服
- 角色权限按功能菜单+按钮粒度配置
- 一个员工可拥有多个角色

## 三、功能点详细设计

### 3.1 账号列表（P1）

#### 功能说明
查看和管理供应商的所有登录账号。

#### 入口路径
左侧导航"系统设置"→"账号列表"

#### 字段说明

| 字段名称 | 字段说明 | 组件类型 |
|---------|---------|---------|
| 账号名 | 登录名（手机号） | 文本 |
| 所属员工 | 关联员工姓名 | 文本 |
| 角色 | 账号角色 | Tag标签 |
| 状态 | 启用/禁用 | 开关 |
| 最后登录 | 最近登录时间 | 时间 |

### 3.2 员工管理（P1）

#### 功能说明
管理供应商的员工信息。

#### 入口路径
左侧导航"系统设置"→"员工管理"

#### 字段说明

| 字段名称 | 字段说明 |
|---------|---------|
| 员工姓名 | 姓名 |
| 手机号 | 联系方式 |
| 岗位 | 岗位说明 |
| 状态 | 在职/离职 |

### 3.3 角色列表（P1）

#### 功能说明
查看供应商的角色列表。

#### 字段说明

| 字段名称 | 字段说明 |
|---------|---------|
| 角色名称 | 角色名 |
| 角色编码 | 系统标识 |
| 员工数 | 绑定该角色的员工数 |
| 操作 | 权限配置按钮 |

### 3.4 权限配置（P1）

#### 功能说明
为角色配置具体的功能操作权限。

#### 交互逻辑
1. 点击"权限配置" → 弹出权限树弹窗
2. 按菜单+按钮粒度勾选权限
3. 保存 → 角色权限更新生效

#### 权限树结构（参考）

```
供应商端
├── 工作台: 数据概览查看
├── 商户信息: 查看、编辑
├── 商品中心
│   ├── 商品列表: 查看、新增、编辑、上下架
│   ├── 库存查询: 查看
│   └── 库存流水: 查看
├── 订单管理
│   ├── 订单列表: 查看、确认、取消、发货
│   ├── 售后管理: 查看、处理补发
│   └── 发货打印: 操作
├── 财务中心
│   ├── 发票管理: 查看、新增、关联、下载
│   ├── 待结算: 查看
│   └── 结算单: 查看
└── 系统设置
    ├── 账号列表: 查看、新增、禁用
    ├── 员工管理: 查看、新增、编辑、离职
    ├── 角色列表: 查看、新增、编辑
    └── 权限配置: 配置
```
