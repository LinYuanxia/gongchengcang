## 一、功能概述

### 1.1 功能定位

财务中心是供应商端的**财务管理模块**，涵盖发票管理、待结算、结算单三大核心能力。

### 1.2 核心概念

| 概念 | 说明 |
|-----|------|
| 进项发票 | 供应商开给工程仓的发票 |
| 待结算 | 已发货但尚未进入结算周期的订单集合 |
| 结算单 | 按结算周期生成的对账单 |

### 1.3 模块范围

| 功能分类 | 主要功能 | 优先级 |
|---------|---------|:------:|
| 发票管理 | 发票列表、新增、关联、详情、下载 | P1-P2 |
| 结算管理 | 待结算列表、结算单列表 | P1 |

---

## 二、业务规则

### 2.1 发票规则

- 一笔订单只能关联一个发票
- 发票已关联不可重复关联
- 发票上传文件格式：PDF/JPG/PNG，大小不超过10MB

### 2.2 结算规则

- 结算周期按自然月
- 已发货且工程仓已确认收货的订单进入待结算
- 结算单生成后不可修改

## 三、功能点详细设计

### 3.1 发票列表（P1）

#### 功能说明
查看供应商已开具的所有发票。

#### 入口路径
左侧导航"财务中心"→"发票列表"

#### 字段说明

| 字段名称 | 字段说明 |
|---------|---------|
| 发票号码 | 发票编号 |
| 关联订单 | 订单编号 |
| 发票金额 | 金额 |
| 开票日期 | 创建时间 |
| 状态 | 有效/无效 |

### 3.2 新增发票（P1）

#### 功能说明
上传发票文件并关联订单。

#### 交互逻辑
1. 上传发票文件 → 自动上传
2. 填写发票号码、发票金额
3. 选择关联订单（下拉选择）
4. 提交 → 生成发票记录

#### 字段说明

| 字段名称 | 组件类型 | 必填 |
|---------|---------|:----:|
| 发票文件 | Upload上传 | ✅ |
| 发票号码 | Input | ✅ |
| 发票金额 | InputNumber | ✅ |
| 关联订单 | Select选择 | ✅ |

### 3.3 关联订单（P1）

#### 功能说明
将已上传的发票与订单关联。

#### 交互逻辑
1. 点击"关联订单" → 弹出订单选择弹窗
2. 选择未关联订单 → 确认
3. 关联成功后更新发票状态

### 3.4 待结算列表（P1）

#### 功能说明
展示已发货但未结算的订单列表。

#### 入口路径
"财务中心"→"待结算"

#### 字段说明

| 字段名称 | 字段说明 |
|---------|---------|
| 订单编号 | 订单号 |
| 工程仓 | 采购方 |
| 订单金额 | 订单总额 |
| 已发货日期 | 发货时间 |
| 结算状态 | 待结算/已结算 |

### 3.5 结算单列表（P1）

#### 功能说明
查看已生成的结算记录。

#### 字段说明

| 字段名称 | 字段说明 |
|---------|---------|
| 结算单编号 | 结算编号 |
| 结算周期 | 起止日期 |
| 结算金额 | 合计金额 |
| 订单数量 | 包含订单数 |
| 状态 | 待确认/已确认 |
