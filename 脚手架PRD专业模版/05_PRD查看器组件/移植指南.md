# PRD查看器 — 移植指南

> 如何将本组件包集成到你的 Vue 3 + Vite 项目。

---

## 第一步：复制文件

将 `05_PRD查看器组件/` 目录下的 `prdData.ts`、`PrdPanel.vue`、`PrdPageViewer.vue` 三个文件复制到你的项目中：

```
你的项目/src/components/
├── PrdPageViewer/
│   ├── prdData.ts          ← 模块注册表
│   ├── PrdPanel.vue        ← 浮窗查看器（可选）
│   └── PrdPageViewer.vue   ← 全页查看器（可选）
```

## 第二步：配置 prdData.ts

```typescript
// 1. 替换 base path
const BASE_PATH = '/你的项目路径'   // 与 vite.config.ts 的 base 一致

// 2. 调整 sourceOptions（单端/多端）
export const sourceOptions = [
  { label: '📌 项目视角', value: 'project' },
  { label: '你的端名', value: '端A' },
  // 单端不需要的删掉
]

// 3. 修改模块列表
// 按 docs/01_PRD/{端名}/ 下的实际文件更新 sourceA_Modules
// 每篇 PRD 文档对应一个 module 条目
```

## 第三步：两种集成方式

### 方式A：全页查看器（推荐，功能完整）

在路由中注册：

```typescript
// router/index.ts
import PrdPageViewer from '@/components/PrdPageViewer/PrdPageViewer.vue'

const routes = [
  {
    path: '/prd',
    name: 'PRD查看器',
    component: PrdPageViewer,
    meta: { title: '产品需求文档' },
  },
]
```

### 方式B：浮窗查看器（嵌入现有页面）

在布局组件中引入：

```vue
<!-- BasicLayout.vue -->
<template>
  <div>
    <router-view />
    <PrdPanel />  <!-- 右下角浮窗按钮 -->
  </div>
</template>

<script setup lang="ts">
import PrdPanel from '@/components/PrdPageViewer/PrdPanel.vue'
</script>
```

## 第四步：放置 PRD 文档

将生成的 PRD markdown 文档放入 `public/prd-docs/`：

```
public/prd-docs/
├── 项目总览.md
├── 系统功能说明书.md
├── 版本迭代记录.md
├── 更新日志.md
├── sourceA/
│   ├── prd.md
│   ├── 01-系统概览与架构.md
│   └── ...
└── sourceB/...
```

文档路径需与 `prdData.ts` 中的 `docUrl` 一致。

## 第五步：验证

| 检查项 | 预期结果 |
|:------|:---------|
| 访问 /prd | 左侧显示模块列表，默认选中第一个 |
| 切换端 | 模块列表切换 |
| 点击模块 | 右侧加载并渲染 MD 文档 |
| Mermaid 图表 | 正确渲染，点击"放大查看"弹窗 |
| 全文搜索 | Ctrl+K 聚焦搜索框，输入关键词匹配 |
| 目录 TOC | 右侧显示标题索引，滚动联动高亮 |

## 注意事项

1. **base 路径一致性** — prdData.ts 中的 `{basePath}`、vite.config.ts 的 `base`、public/ 下的 prd-docs，三者路径必须统一
2. **Arco Design** — 组件依赖 Arco Design Vue 的组件库，如未使用需替换或安装
3. **单系统简化** — 如果只有1个端，直接修改 PrdPageViewer.vue 中的 `activeModules` computed 为固定引用，去掉 `switch-case`
4. **prd.png** — 浮窗按钮的图标放在 `public/prd.png`，可自行替换
