# 系统操作手册模板（系统说明书）

> 适用：PC端操作手册 / APP端操作手册
> 面向：最终用户（非技术人员）
> 版本：v1.0

---

## 模板说明

### 使用方式

1. 复制本模板到 `02_系统操作手册/{端名}/`
2. 按章节填充内容
3. 截图统一放入 `images/` 目录，分模块存放
4. 错误截图放入 `images/error/` 子目录

### 章节概览

| 章节 | 内容 | 截图需求 |
|:----|:-----|:--------|
| 1. 操作前必读 | ⚠️ 核心规则5条以上 | 无 |
| 2. 快速入门 | 登录+第一个核心任务 | 3~5张 |
| 3. 功能详解 | 按模块逐一说明 | 每模块2~5张 |
| 4. 典型工作场景 | 跨模块真实场景 | 3~5张场景截屏 |
| 5. 常见问题FAQ | 14条以上Q&A | 无 |
| 6. 常见错误提示 | 错误→含义→处理 | 6~10张错误截图 |
| 7. 注意事项 | 结构化提示 | 无 |
| 8. 联系支持 | 联系方式表 | 无 |
| 附录A | 文档版本记录 | 无 |
| 附录B | 截图采集清单 | — |
