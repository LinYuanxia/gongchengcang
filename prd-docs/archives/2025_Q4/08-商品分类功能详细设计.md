# 工程材料管理采供一体化平台 - 商品分类功能详细设计

> 版本：v1.0  
> 文档状态：评审中  
> 所属模块：商品中心 - 商品分类

---

## 一、功能概述

### 1.1 功能定位

商品分类管理是平台商品体系的基础核心功能，负责建立和维护商品的分类体系，为商品的组织、搜索、展示提供基础支撑。分类采用树形结构组织，支持三级分类层级，为平台的商品管理提供清晰的分类体系。

### 1.2 目标用户

- 平台管理员
- 平台运营人员

---

## 二、业务规则

### 2.1 分类层级规则

- **层级限制**：最多支持三级分类
  - 一级分类：顶级分类，无父分类
  - 二级分类：隶属于一级分类
  - 三级分类：隶属于二级分类
- **树形结构**：分类采用树形结构组织，理论上支持无限级分类，但建议最多使用三级分类以保证用户体验
- **层级校验**：创建分类时校验分类层级，不允许超过三级

### 2.2 分类编码规则

- **编码规则**：使用层级编码方式
  - 一级分类：3位数字，如 `001`、`002`、`003`
  - 二级分类：6位数字，父分类编码（3位）+ 子分类编码（3位），如 `001001`、`001002`
  - 三级分类：9位数字，父分类编码（6位）+ 子分类编码（3位），如 `001001001`、`001001002`
- **编码唯一性**：分类编码在系统中全局唯一
- **编码自动生成**：新增分类时，系统自动根据父分类和序号生成分类编码
- **编码不可修改**：分类编码一旦创建不可修改

### 2.3 状态约束规则

- **分类状态**：分类有两种状态
  - 启用：分类正常使用，可以在该分类下创建商品
  - 禁用：分类暂停使用，不能在该分类下创建新商品
- **状态级联**：
  - 父分类禁用时，所有子分类自动禁用
  - 父分类启用时，子分类状态保持不变（需手动启用）
- **状态校验**：禁用的分类下不能创建新商品，已有的商品不受影响

### 2.4 删除约束规则

- **删除校验**：删除分类前必须检查是否有关联数据
  - 检查是否有子分类：有子分类的分类不能直接删除，需先删除子分类
  - 检查是否有关联商品：有关联商品的分类不能直接删除，需先取消商品关联
- **删除提示**：删除时根据校验结果给出明确的提示信息
  - 有子分类时提示："该分类下有子分类，请先删除子分类"
  - 有关联商品时提示："该分类下有关联商品，请先取消商品关联"
- **级联删除**：不支持级联删除，需逐级删除

### 2.5 排序规则

- **同级排序**：同一级别的分类支持排序
- **排序方式**：
  - 手动拖拽排序：在分类树中拖拽分类调整顺序
  - 手动输入排序值：在分类编辑页面输入排序值
- **排序值**：使用整数排序值，数值越小排序越靠前
- **排序重排**：拖拽排序后自动重新计算排序值

## 三、功能点详细设计

### 3.1 分类列表展示

#### 功能说明
以树形结构展示商品分类列表，支持分页和搜索

#### 交互逻辑
1. 页面加载时默认展示一级分类
2. 点击分类名称左侧的展开/折叠按钮可展开/折叠子分类
3. 分类树节点显示信息：
   - 分类名称（最多显示10字，超出显示省略号）
   - 分类编码（灰色小字）
   - 状态标签（启用/禁用）
   - 关联商品数量（SPU数量/SKU数量）
4. 鼠标悬停在分类节点上时显示操作按钮
5. 支持分页加载，当分类数量较多时采用懒加载方式

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 展示方式 |
|---------|---------|---------|---------|
| 分类名称 | 分类的名称，最多10字 | VARCHAR | 主标题展示 |
| 分类编码 | 分类的唯一编码，按层级规则生成 | VARCHAR | 灰色小字展示 |
| 分类层级 | 分类的层级（1/2/3） | INT | 不展示，用于逻辑判断 |
| 状态 | 分类状态（启用/禁用） | TINYINT | 标签展示 |
| 排序 | 分类的排序值 | INT | 不展示，用于排序 |
| SPU数量 | 该分类下的SPU商品数量 | INT | 数字展示 |
| SKU数量 | 该分类下的SKU商品数量 | INT | 数字展示 |
| 创建时间 | 分类的创建时间 | DATETIME | 不展示 |
| 更新时间 | 分类的更新时间 | DATETIME | 不展示 |

#### 界面元素
- 分类树容器
- 展开/折叠按钮
- 搜索框
- 新增分类按钮
- 批量操作按钮
- 分页控件（懒加载时使用）

---

### 3.2 搜索分类

#### 功能说明
在左侧搜索框输入关键词，支持模糊搜索分类信息，搜索结果在分类树中高亮显示

#### 交互逻辑
1. 搜索框位于页面左侧顶部，默认显示占位文字"搜索分类名称或编码"
2. 输入关键词后实时触发搜索（支持防抖，延迟300ms）
3. 支持按回车键触发搜索
4. 搜索结果处理：
   - 匹配的分类节点在树中高亮显示
   - 匹配的分类节点自动展开父分类
   - 不匹配的分类节点自动折叠
   - 无匹配结果时显示"暂无搜索结果"提示
5. 清空搜索框时恢复分类树的原始展示状态

#### 字段说明
| 字段名称 | 字段说明 | 匹配规则 |
|---------|---------|---------|
| 分类名称 | 按分类名称模糊匹配 | 包含关键词即匹配 |
| 分类编码 | 按分类编码精确匹配或前缀匹配 | 完全匹配或前缀匹配 |

#### 界面元素
- 搜索输入框
- 搜索图标
- 清空按钮（输入内容后显示）
- 搜索结果提示（无结果时显示）

---

### 3.3 新增分类

#### 功能说明
通过弹窗表单填写分类信息，创建新的商品分类

#### 交互逻辑
1. 点击"新增分类"按钮弹出新增分类弹窗
2. 表单包含以下字段：
   - **分类名称**：
     - 必填项，带*号标识
     - 输入框，最多10个字符
     - 实时校验：输入时显示剩余字符数（如"5/10"）
     - 失焦时校验唯一性（同一父分类下名称唯一）
   - **上级分类**：
     - 下拉选择框，默认为空（表示一级分类）
     - 下拉框以树形结构展示已启用的分类
     - 最多只能选择到二级分类（因为三级分类不能有子分类）
     - 选择后自动计算分类层级
   - **排序**：
     - 输入框，默认值为当前同级分类的最大排序值+1
     - 只能输入正整数
     - 不填则使用默认值
   - **状态**：
     - 单选按钮组，默认选中"启用"
     - 选项：启用 / 禁用
     - 禁用状态时给出提示："禁用后该分类下不能创建新商品"
3. 点击"确定"按钮：
   - 前端校验表单完整性
   - 调用后端接口创建分类
   - 系统自动生成分类编码（根据父分类和序号）
   - 创建成功后关闭弹窗，刷新分类树
   - 弹出成功提示："分类创建成功"
4. 点击"取消"按钮或弹窗外部关闭弹窗，表单数据不保存

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 | 约束 | 默认值 |
|---------|---------|---------|------|--------|
| 分类名称 | 分类名称，最多10字 | VARCHAR(10) | 必填，同一父分类下唯一 | - |
| 上级分类 | 父分类ID，为空表示一级分类 | BIGINT | 可选，最多选择到二级分类 | 空 |
| 排序 | 排序值，正整数 | INT | 可选 | 当前同级最大排序值+1 |
| 状态 | 分类状态 | TINYINT | 必填 | 1（启用） |

#### 界面元素
- 弹窗标题：新增分类
- 表单容器
- 分类名称输入框（带字符计数）
- 上级分类下拉树选择器
- 排序输入框
- 状态单选按钮组
- 确定按钮
- 取消按钮
- 表单验证提示

---

### 3.4 编辑分类

#### 功能说明
编辑已存在的分类信息，操作与新增分类一致

#### 交互逻辑
1. 在分类树节点的操作按钮中点击"编辑"按钮
2. 弹出编辑分类弹窗，表单字段与新增分类一致
3. 表单自动填充当前分类的原有数据
4. 字段规则：
   - 分类名称：可修改，规则同新增
   - 上级分类：可修改，但有以下限制：
     - 不能选择自己的子分类作为父分类
     - 修改后分类层级不能超过三级
   - 排序：可修改，规则同新增
   - 状态：可修改，规则同新增
5. 点击"确定"按钮：
   - 前端校验表单完整性
   - 调用后端接口更新分类
   - 更新成功后关闭弹窗，刷新分类树
   - 弹出成功提示："分类更新成功"
6. 点击"取消"按钮或弹窗外部关闭弹窗，修改不保存

#### 字段说明
同新增分类字段说明

#### 界面元素
- 弹窗标题：编辑分类
- 其他界面元素同新增分类

---

### 3.5 新增子分类

#### 功能说明
为当前分类创建子分类，快速建立分类层级关系

#### 交互逻辑
1. 在分类树节点的操作按钮中点击"新增子分类"按钮
2. 弹出新增分类弹窗，表单字段与新增分类一致
3. 表单自动填充：
   - 上级分类：自动选中当前点击的分类
   - 上级分类选择框禁用，不可修改
   - 排序：默认值为当前分类的子分类的最大排序值+1
4. 分类层级限制：
   - 如果当前分类是三级分类，则"新增子分类"按钮禁用，并给出提示："三级分类下不能创建子分类"
5. 其他交互逻辑同新增分类

#### 字段说明
同新增分类字段说明，上级分类字段自动填充且不可修改

#### 界面元素
- 弹窗标题：新增子分类
- 其他界面元素同新增分类
- 上级分类选择框禁用状态

---

### 3.6 删除分类

#### 功能说明
删除指定分类，删除前需要校验是否有关联数据

#### 交互逻辑
1. 在分类树节点的操作按钮中点击"删除"按钮
2. 弹出确认删除对话框
3. 系统执行删除前校验：
   - **校验1：检查是否有子分类**
     - 如果有子分类，对话框显示警告信息："该分类下有 {数量} 个子分类，请先删除子分类"
     - "确认"按钮置灰，不可点击
     - 显示"取消"按钮
   - **校验2：检查是否有关联商品**
     - 如果有关联商品，对话框显示警告信息："该分类下有关联的商品（SPU: {数量}个，SKU: {数量}个），请先取消商品关联"
     - "确认"按钮置灰，不可点击
     - 显示"取消"按钮
   - **校验3：无关联数据**
     - 对话框显示确认信息："确定要删除分类「{分类名称}」吗？删除后不可恢复。"
     - 显示"确认"和"取消"按钮
4. 点击"确认"按钮：
   - 调用后端接口删除分类
   - 删除成功后关闭对话框，刷新分类树
   - 弹出成功提示："分类删除成功"
5. 点击"取消"按钮或弹窗外部关闭对话框，不执行删除

#### 字段说明
无额外字段，主要是校验逻辑

#### 界面元素
- 确认删除对话框
- 警告信息展示区域
- 确认按钮（校验通过时可用）
- 取消按钮

---

### 3.7 分类拖拽排序

#### 功能说明
在分类树中支持拖拽调整分类的显示顺序

#### 交互逻辑
1. 分类树节点支持拖拽操作
2. 拖拽规则：
   - 只能在同一父分类下的兄弟节点之间拖拽排序
   - 不能拖拽到其他父分类下（如需修改父分类请使用编辑功能）
   - 三级分类不能拖拽到一级分类下
3. 拖拽视觉效果：
   - 拖拽时显示拖拽占位符
   - 拖拽节点半透明显示
   - 可放置区域高亮显示
4. 拖拽完成后：
   - 系统自动重新计算所有兄弟节点的排序值
   - 调用后端接口批量更新排序值
   - 刷新分类树
   - 弹出成功提示："排序更新成功"
5. 支持取消拖拽：按ESC键或拖拽到无效区域取消拖拽操作

#### 字段说明
| 字段名称 | 字段说明 | 数据类型 |
|---------|---------|---------|
| 排序 | 分类的排序值 | INT |

#### 界面元素
- 拖拽占位符
- 可放置区域高亮
- 拖拽节点半透明效果

---

### 3.8 查看分类关联商品

#### 功能说明
点击分类树节点上的SPU/SKU数量，跳转到商品列表页面查看该分类下的所有商品

#### 交互逻辑
1. 在分类树节点上显示SPU数量和SKU数量标签
2. 点击SPU数量标签：
   - 跳转到商品SPU列表页面
   - 自动筛选该分类下的所有SPU商品
   - 页面顶部显示筛选条件："分类：{分类名称}"
3. 点击SKU数量标签：
   - 跳转到商品SKU列表页面
   - 自动筛选该分类下的所有SKU商品
   - 页面顶部显示筛选条件："分类：{分类名称}"
4. 支持返回：在商品列表页面提供返回按钮，可返回分类管理页面

#### 字段说明
无额外字段，主要是页面跳转和筛选逻辑

#### 界面元素
- SPU数量标签（可点击）
- SKU数量标签（可点击）
- 商品列表页面筛选条件展示

## 四、数据校验规则

### 4.1 分类名称校验
- 必填校验：分类名称不能为空
- 长度校验：最多10个字符
- 唯一性校验：同一父分类下的分类名称不能重复
- 特殊字符校验：不能包含特殊字符（支持中文、英文、数字、下划线、横线）

### 4.2 上级分类校验
- 层级校验：选择上级分类后，分类层级不能超过三级
- 循环校验：不能选择自己的子分类作为父分类
- 状态校验：只能选择启用状态的分类作为上级分类

### 4.3 排序值校验
- 必须是正整数
- 可以为空，为空时使用默认值

### 4.4 删除校验
- 子分类校验：有子分类的分类不能删除
- 商品关联校验：有关联商品的分类不能删除

## 五、异常处理

### 5.1 表单验证异常
- 实时显示验证错误信息
- 提交时再次校验，阻止无效提交

### 5.2 网络异常
- 操作失败时显示错误提示
- 提供重试按钮
- 失败时保持表单数据，避免重复输入

### 5.3 权限异常
- 无操作权限时隐藏操作按钮
- 尝试操作时显示权限不足提示

## 八、接口设计

### 8.1 获取分类树
**接口路径**：`GET /api/product/category/tree`  
**接口说明**：获取商品分类树结构  
**请求参数**：无  
**响应数据**：分类树结构的JSON数据

### 8.2 搜索分类
**接口路径**：`GET /api/product/category/search`  
**接口说明**：搜索分类  
**请求参数**：
- `keyword`：搜索关键词（必填）
**响应数据**：匹配的分类列表

### 8.3 新增分类
**接口路径**：`POST /api/product/category`  
**接口说明**：新增分类  
**请求参数**：
- `categoryName`：分类名称（必填）
- `parentId`：父分类ID（可选）
- `sort`：排序值（可选）
- `status`：状态（必填）
**响应数据**：新增的分类信息

### 8.4 编辑分类
**接口路径**：`PUT /api/product/category/{id}`  
**接口说明**：编辑分类  
**请求参数**：
- `categoryName`：分类名称（必填）
- `parentId`：父分类ID（可选）
- `sort`：排序值（可选）
- `status`：状态（必填）
**响应数据**：更新后的分类信息

### 8.5 删除分类
**接口路径**：`DELETE /api/product/category/{id}`  
**接口说明**：删除分类  
**请求参数**：无  
**响应数据**：删除结果

### 8.6 更新排序
**接口路径**：`PUT /api/product/category/sort`  
**接口说明**：批量更新分类排序  
**请求参数**：
- `sorts`：排序信息数组，包含分类ID和排序值（必填）
**响应数据**：更新结果

### 8.7 检查删除条件
**接口路径**：`GET /api/product/category/{id}/check-delete`  
**接口说明**：检查分类是否可以删除  
**请求参数**：无  
**响应数据**：
- `canDelete`：是否可以删除（true/false）
- `hasChildren`：是否有子分类
- `hasProducts`：是否有关联商品
- `spuCount`：SPU数量
- `skuCount`：SKU数量

## 九、总结

本商品分类功能详细设计完整描述了商品分类管理的所有功能点，包括：

1. **分类列表展示** - 以树形结构展示分类，支持折叠展开
2. **搜索分类** - 支持按名称或编码搜索，结果高亮展示
3. **新增分类** - 通过弹窗表单创建新分类，自动生成分类编码
4. **编辑分类** - 修改分类信息，操作与新增一致
5. **新增子分类** - 为当前分类创建子分类，快速建立层级
6. **删除分类** - 删除前校验是否有子分类或关联商品
7. **分类拖拽排序** - 支持在同一父分类下拖拽调整顺序
8. **查看分类关联商品** - 点击SPU/SKU数量跳转到商品列表

每个功能点都包含了详细的功能说明、交互逻辑、字段说明和界面元素，同时明确了业务规则、数据校验规则、异常处理和权限控制，为商品分类功能的开发提供了完整的设计指导。
