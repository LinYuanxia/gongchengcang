## 一、功能概述

### 1.1 功能定位

工作台与商户中心是供应商端的**核心入口模块**，工作台提供运营数据概览看板，商户中心管理供应商的主体信息与合同。

### 1.2 核心概念

| 概念 | 说明 | 示例 |
|:----|------|------|
| 数据概览 | 供应商日常运营的核心指标看板 | 订单总金额/待发货/待结算 |
| 主体信息 | 供应商的工商注册信息、联系方式等基础档案 | 公司全称/信用代码/营业执照 |
| 合同列表 | 供应商与平台的合作协议列表（只读） | 供货框架协议 |

### 1.3 目标用户

- **管理员**（核心用户）：查看运营概览、编辑主体信息
- **业务员**：查看数据概览和合同
- **财务**：查看数据概览

### 1.4 模块范围

| 功能分类 | 主要功能 | 涉及角色 |
|:--------|---------|---------|
| 工作台 | 数据概览（指标卡片+趋势） | 管理员、业务员、财务 |
| 商户信息 | 查看/编辑主体信息 | 管理员 |
| 商户信息 | 合同列表查看 | 管理员、业务员 |

## 二、核心设计原则

> **工作台遵循"轻量概览"原则，商户中心遵循"只读+编辑分离"原则。**

### 2.1 数据轻量化原则

- 工作台只展示关键运营指标，不做明细数据下钻
- 每个指标卡片独立加载，互不影响（容错隔离）
- 默认展示"今日"数据，切换时间范围时重新请求

### 2.2 只读与编辑分离原则

- 主体信息默认只读展示，只有管理员可见「编辑」按钮
- 编辑模式下实时校验字段，保存后自动恢复只读状态
- 可编辑字段仅限于联系人/手机号/邮箱，工商信息不可编辑

## 三、业务规则

### 3.1 数据概览规则

- **默认时间范围**：今日（00:00-23:59）
- **可切换时间**：今日、本周、本月
- **指标自动刷新**：页面加载时获取，暂不做轮询
- **空数据展示**：无数据时显示"暂无数据"，不展示0值

### 3.2 主体信息规则

- 主体信息由平台端审核录入，供应商端仅可查看和编辑部分字段
- 可编辑字段：联系人姓名、联系人手机号、联系邮箱
- 不可编辑字段：公司全称、统一社会信用代码、法人代表等工商信息

## 六、功能点详细设计

### 6.1 数据概览（P1）

#### 交互逻辑

1. 页面加载：请求统计接口 → 渲染指标卡片
2. 切换时间维度：点击今日/本周/本月 → 重新请求对应数据
3. 点击指标卡片：跳转到对应模块的列表页

#### 原子字段定义

| 字段 | 必填 | 来源 | 展示规则 | 默认值 |
|:----|:----|:----:|:----|:--------|:-----:|
| 订单总金额 | 否 | 统计接口 | 数字卡片(大号) | 加载中... |
| 订单数 | 否 | 统计接口 | 数字卡片 | 加载中... |
| 待发货 | 否 | 统计接口 | 数字卡片(黄色) | 加载中... |
| 待结算金额 | 否 | 统计接口 | 数字卡片 | 加载中... |
| 在售商品数 | 否 | 统计接口 | 数字卡片 | 加载中... |

#### 边界情况覆盖

| 场景 | 处理逻辑 | 提示文案 |
|:----|:--------|---------|
| 统计接口超时 | 卡片显示"--" | Toast提示"数据加载失败，请稍后重试" |
| 无统计数据 | 卡片显示0 | - |

---

### 6.2 主体信息查看（P1）

#### 交互逻辑

1. 页面加载：请求主体信息接口 → 渲染详情展示
2. 点击"编辑"按钮 → 进入编辑模式（仅管理员可见按钮）

#### 原子字段定义

| 字段 | 必填 | 来源 | 展示规则 |
|:----|:----|:----:|:----|:--------|
| 公司全称 | 是 | 平台录入 | 主标题 |
| 统一信用代码 | 是 | 平台录入 | 等宽字体 |
| 法人代表 | 是 | 平台录入 | 文本 |
| 联系人 | 否 | 供应商填写 | 文本 |
| 联系电话 | 否 | 供应商填写 | 脱敏：138****1234 |
| 联系邮箱 | 否 | 供应商填写 | 邮箱格式 |

---

### 6.3 主体信息编辑（P1）

#### 交互逻辑

1. 点击编辑 → 表单字段变为可编辑状态
2. 修改后点击"保存" → 提交修改
3. 保存成功：Toast提示"修改成功"
4. 保存失败：Toast提示"修改失败：{原因}"

#### 原子字段定义

| 字段 | 必填 | 来源 | 校验规则 | 展示规则 |
|:----|:----|:----:|:----|:--------|:--------|
| 联系人姓名 | 是 | 表单输入 | 最长20字符 | Input框 |
| 联系人手机号 | 是 | 表单输入 | ^1[3-9]\d{9}$ | Input框+格式校验 |
| 联系邮箱 | 否 | 表单输入 | 邮箱格式校验 | Input框 |

---

### 6.4 合同列表（P2）

查看供应商与平台的合作协议列表（只读）。列表展示：合同名称/合同编号/有效期/签署日期/附件下载。

## 七、异常处理汇总表

| 异常场景 | 前端处理 | 提示文案 |
|:--------|:--------|---------|
| 统计接口超时 | 卡片显示"--" | "数据加载失败，请稍后重试" |

| 联系电话格式错误 | 表单标红 | "请输入正确的手机号" |
| 合同加载失败 | Toast | "合同加载失败，请重试" |
